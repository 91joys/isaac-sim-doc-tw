物理模擬（Physics）#

對應官方 Physics 章節的中文導覽。Isaac Sim 6.0 支援 PhysX 與 Newton 兩套可切換的物理後端，並提供一系列檢視與除錯工具。

核心概念#

  • Physics Scene：物理模擬的容器 prim，定義重力、求解器、GPU 動力學等全域設定（Python 建立方式見《完整文檔》6.4 節的 PhysxScene）。
  • 剛體與碰撞分離：Rigid Body（質量、受力）與 Collider（碰撞形狀）是獨立的 API，可自由組合——會擋不會掉、會掉不會擋、或兩者兼具。
  • Articulation：以關節連接的多剛體結構（機器人），由專用求解器處理，穩定性遠優於逐一約束。
  • 物理只在時間軸播放時步進——這條規則貫穿所有工作流程。

常用 Python 操作速覽#

（完整片段見《完整文檔》第 6 章與官方 Scene Setup Snippets）

# 物理場景與重力
from isaacsim.core.simulation_manager import PhysxScene, SimulationManager
scene = PhysxScene("/World/physicsScene")
scene.set_gravity([0.0, 0.0, -9.81])
scene.set_solver_type("TGS")            # 求解器
SimulationManager.set_device("cpu")     # CPU / GPU 物理

# 碰撞近似
from isaacsim.core.experimental.prims import GeomPrim
geom = GeomPrim("/World/Cube", apply_collision_apis=True)
geom.set_collision_approximations(["convexHull"])   # 或 convexDecomposition / none

# 物理材質（摩擦係數）
from isaacsim.core.experimental.materials import RigidBodyMaterial
mat = RigidBodyMaterial("/World/PhysicsMaterials", static_frictions=[0.5], dynamic_frictions=[0.5])
geom.apply_physics_materials(mat)

# 查詢目前使用的物理引擎
SimulationManager.get_active_physics_engine()   # "physx" 或 newton

接觸力與場景查詢#

  • 接觸力：以 RigidPrim 的 contact filter 參數（contact_filter_paths、max_contact_count）建立後，可用 get_net_contact_forces()、get_contact_force_matrix()、get_friction_data()、get_contact_force_data() 取得淨力、力矩陣、摩擦與逐接觸點資料（官方完整範例）。
  • 場景查詢：get_physx_scene_query_interface() 的 raycast_closest（射線）與 overlap_box / overlap_sphere（重疊）——用法見《Script Editor 教學》第 9 課。

調校建議#

  • 機器人抖動／飛走：先查關節驅動的 stiffness/damping（Gain Tuner、關節調校教學）。
  • 穿透問題：開 CCD（set_enabled_ccd(True)）、提高求解迭代、改用更緊的碰撞近似。
  • 效能：見 Isaac Sim 效能優化手冊。
