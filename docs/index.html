Isaac Sim 與 Newton 的關係

Isaac Sim 是完整的機器人「模擬器應用」，Newton 是專注於物理計算的「物理引擎」。Isaac Sim 6.0 起支援以 Newton 作為可切換的物理後端，Isaac Lab 亦提供實驗性 Newton 整合——兩者互補而非取代。

Isaac SimNewton Physics
定位全功能機器人模擬器（渲染、感測器、ROS 2、合成資料）GPU 加速物理引擎（模擬計算核心）
主要用途數位孿生、感測器模擬、sim-to-real 工作流程機器人學習（RL）、可微分模擬、多物理研究
物理後端PhysX 與 Newton 可切換自身即引擎：MuJoCo、XPBD、VBD、Featherstone 等 8 種求解器
使用介面GUI ＋ Python（Kit / Omniverse）純 Python API（建構於 NVIDIA Warp）
維護者NVIDIALinux Foundation 專案（Disney Research、Google DeepMind、NVIDIA）
