Kit-CAE 1.5 舊版文檔#

對應官方 Kit-CAE 1.5（Legacy） 章節。記錄原始的 Kit-CAE 1.5 工作流程。

使用 Kit-CAE 2.0 的讀者：請改看新版指南（總覽文檔起）。本頁僅供維護以 1.5 建立的舊專案時參考。2.0 是一次重大演進，因此獨立成專屬文檔，而非併入版本說明。

1. 涵蓋主題#

舊版文檔包含三大主題：

Get StartedKit-CAE 1.5 使用者的基礎入門指引。
Create a Mesh-Based Dataset建立以網格幾何為結構的資料集。

2. 與 2.0 的差異#

索引頁未詳列逐項功能比較，但文檔明確暗示：Kit-CAE 2.0 相對 1.5 是一次重大演進，值得獨立、專屬的文檔，而非以版本說明整合帶過。2.0 的主要方向包括：

  • 介面明確區分 Sources／Operators／Flow 三類實體。
  • 逐運算子的裝置選擇、執行開關、欄位分量選擇與屬性鎖定。
  • 新增 OpenFOAM 直接匯入、EnSight 多面體支援、VTK 讀取加速。
  • GPU 加速的統計計算與 CAE Insights、AOT 預編譯快取。
  • 預設改為 Z-up，對齊 CAE 業界慣例。
相容性：在 2.0 中，舊版 CAE 選單預設停用，但仍可用於較早建立的舞台，並可於 Edit > Preferences > CAE 的 Legacy Settings 重新啟用。
