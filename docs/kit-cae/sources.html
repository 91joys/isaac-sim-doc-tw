建立來源（Sources）#

對應官方 Create Sources 章節。來源是輔助運算子工作的「不可變幾何」，本身不做運算。

1. 什麼是來源#

來源（Sources）是 Kit-CAE 中的不可變輔助實體。與運算子不同，它們：

  • 重載舞台時不會重新執行；
  • 是沒有套用任何 CAE 運算子 API schema 的普通 USD prim。

換句話說，來源就是一些純粹的幾何（框、球），用來當作運算子的「感興趣區域」或「種子」。

2. 如何建立來源#

  1. 在 Stage 面板選取一個 prim。
  2. 右鍵，選 Create > CAE Sources。
  3. 選擇想要的來源類型。
選項依實體型別而定：所有資料集 prim 預設都提供邊界框與單位球；而像切面（slice）這類特化來源只在特定實體上出現（例如切面來源會出現在體積運算子的結果上）。

3. 邊界框（Bounding Box）#

邊界框用來視覺化空間範圍，同時也是限制運算子運算域的感興趣區域（ROI）。特性：

  • 從資料集節點建立。
  • 建立時取用資料集當下的 bounds，之後即斷開連結（不再隨資料變動）。
  • 可重新命名、變換（transform）與縮放。
  • 支援自訂尺寸——例如 scale by X: 2, Y: 1, Z: 1，把框往後延伸以涵蓋車輛尾流（wake）區域。
  • 兼具雙重用途：視覺參考＋運算邊界（供串流線與體積渲染運算子使用）。

4. 單位球（Unit Sphere）#

單位球為串流線運算子與流場發射器提供種子幾何（seed geometry）。特性：

  • 球體每個頂點都成為一條串流線的起點，或一個粒子的發射源。
  • 依所選物件的 bounds 自動定位與定尺寸。
  • 可自由移動，用來檢視不同流場區域（進氣口流動、局部渦流循環…）。
  • 可在運算子設定中，用自訂幾何取代這顆球。

5. 試試看：為串流線與體積鋪路#

  1. 選取汽車資料集節點，右鍵 Create > CAE Sources > Bounding Box。
  2. 把邊界框 scale X: 2，讓它往車後延伸，涵蓋尾流區。
  3. 再建立一個 Unit Sphere，移動到進氣或側面感興趣的位置。
  4. 這兩個來源接著會被串流線運算子拿來當種子與 ROI。
