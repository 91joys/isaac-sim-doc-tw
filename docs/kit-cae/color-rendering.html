色彩對應與渲染#

對應官方 Color Mapping and Rendering 章節。有視覺輸出的運算子都能透過欄位選擇、色彩表與屬性鎖定做純量著色。

1. 欄位選擇（著色依據）#

  1. 在 Stage 面板選取運算子 prim。
  2. 在 Properties 找到 Colors [Field Selection]，點 Add Target。
  3. 瀏覽並選取想要的欄位。

Kit-CAE 支援三種著色方式：

欄位型態著色方式
單分量欄位（shape N,1）直接純量著色，如壓力、溫度。
三分量欄位（shape N,3）以計算出的向量模長（magnitude）著色，例如速度 X／Y／Z 以整體速率著色。
三個獨立單分量欄位用 Ctrl／Shift 多選各分量，Kit-CAE 會把它們視為一個向量並以模長著色。

2. 色彩表（Colormaps）#

Kit-CAE 內建色彩表：afmhot、cividis、gist_gray、gist_rainbow（預設）。

Faces／Streamlines／Points／Glyphs

  • 展開運算子 prim → Materials → ScalarColor → Shader。
  • 調 domain 欄位設定色彩範圍。
  • 用 lut 欄位選擇不同色彩表。

Volume／Slice

  • 展開運算子 → Material → Colormap。
  • 在此設定色彩範圍與傳遞函數（transfer function）。

3. 色彩範圍行為（時變資料）#

對時變資料，色彩範圍可有三種反應方式，讓動畫過程中色階穩定：

模式行為
Clamp鎖定在目前的數值範圍。
Grow隨時間步出現新值而擴張範圍。
Do nothing維持範圍不變。

也可直接在 colormap 手動設定 domain 值，做精確控制。

4. 鎖定機制（Lock）#

運算子重算時可能把屬性重設回預設值。要避免辛苦調好的設定被覆蓋：

  1. 把屬性設成想要的值。
  2. 右鍵該屬性欄位，選 Locks > Lock。
  3. 鎖定值即可穿越重算而保留。

要恢復自動更新，右鍵選 Locks > Unlock 即可。

典型用法：移動串流線種子球時運算子會重算——把 domain（色彩範圍）鎖定，串流線就能在重算時維持一致的配色。

5. Shader 參數#

體積渲染使用 MDL shader，參數透過 API schema 控制，提供明確的渲染行為控制。自訂 shader 的屬性名稱可能不同。多數情況用預設值即可；進階使用者可在 Properties 面板的 Raw Properties 檢視所有 shader 參數。

6. 試試看：調校色彩對應#

體積（Volume）

  1. 隱藏其他運算子，只留 Volume 與 Faces 可見。
  2. 展開 Volume 運算子 → Material → Colormap。
  3. 設範圍 min 0、max 30。
  4. 把 Domain Boundary Mode 設為 clampToTransparent（讓超出範圍的值完全透明）。
  5. 調整透明度曲線控制不透明度。
  6. Lock 住 Colormap，避免重算重設設定。

體積切面（Volume Slice）

  1. 顯示 VolumeSlice，可選擇隱藏 Volume。
  2. 選取切面平面 prim，用變換 gizmo 穿過幾何。
  3. 展開 VolumeSlice → Material → Colormap。
  4. 設範圍 min 0、max 40；Domain Boundary Mode 設 clampToEdge。

串流線（Streamlines）

  1. 顯示 Streamlines 與 Faces。
  2. 選 Streamlines 運算子，捲到 Colors [Field Selection]，點 Add Target。
  3. 選 Pressure 之類的欄位，或 Shift 多選 Velocity 分量。
  4. 展開 Streamlines → Materials → ScalarColor → Shader。
  5. 調 domain 欄位得到有意義的色彩範圍。
  6. 用 lut 資料夾圖示瀏覽其他色彩表。
  7. Lock domain 欄位，讓串流線重算時保持配色。移動種子球即可重定位串流線，同時保留鎖定的色彩設定。
