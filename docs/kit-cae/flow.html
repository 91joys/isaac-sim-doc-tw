流場視覺化#

對應官方 Flow Visualization 章節。用 NVIDIA Flow 把速度場變成煙霧般的粒子動畫。

1. 核心概念#

Flow 視覺化提供粒子動畫式的流場視覺化，底層採 NVIDIA Flow 技術。這套 GPU 加速系統沿著由速度場推導出的路徑推進粒子，做出直覺、煙霧般的效果，揭示流體移動型態。

適用時機：能呈現流向、回流與紊流型態，特別適合探索複雜的 3D 流場結構，以及向非技術觀眾展示成果。

要更精確的流場型態分析？官方建議改用 Streamlines 運算子——Flow 偏視覺呈現，Streamlines 偏定量分析。

2. 設定流程（四步驟）#

步驟 1：建立 Flow 環境

流場環境是所有視覺化元素的容器。右鍵資料集 prim → Create > CAE Flow > Environment。會產生一個 FlowSimulate prim，多數情況建議使用 layer 0。

步驟 2：建立燃料噴射球（Fuel Injector Sphere）

這個元件控制粒子的發射點。右鍵資料集節點 → Create > CAE Flow > Fuel Injector Sphere。產生的發射器球 prim 可自由移動，從域內不同位置探索流動行為。

步驟 3：設定資料集注入器（DataSet Injector）

資料集注入器把流場視覺化連到模擬資料。右鍵 → Create > CAE Flow > DataSet Injector，再設定屬性：

  • Velocity fields（速度欄位）：Add Target 加入速度分量欄位。
  • Temperature fields（溫度欄位）：Add Target 加入純量欄位，做額外資料表現。

步驟 4：啟動動畫

兩種方式啟動流場：在 FlowSimulate prim 勾選 Force Simulate，或按時間軸的 Play 鍵。

3. 試試看：車輛空氣力學流場#

把上述概念套到車輛流場分析。關鍵一步是把 Inflate Bounds (%) 設為 100，擴大體素化區域，避免粒子逸出分析範圍。

  1. 右鍵汽車資料集 → Create > CAE Flow > Environment。
  2. 建立 Fuel Injector Sphere，放到進氣或車頭位置。
  3. 建立 DataSet Injector，Velocity 加入速度欄位。
  4. 把 Inflate Bounds (%) 設 100。
  5. 勾 Force Simulate 或按 Play，觀察煙霧般的流動沿車身展開。
