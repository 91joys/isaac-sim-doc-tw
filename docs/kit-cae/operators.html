建立運算子（Operators）#

對應官方 Create Operators 章節。運算子是 Kit-CAE 的視覺化主力——動態、即時、GPU 加速。

1. 什麼是運算子#

運算子（Operators）是動態視覺化實體，會在重載舞台時即時重算，從來源資料當場產生結果。每個運算子 prim 都套用了 Omni CAE Operator API。

建立方式：在 Stage 面板右鍵一個 prim，選 Create > CAE Operators。

2. 共通屬性#

所有運算子在 Properties 面板的 CAE 群組下都有這些可調設定：

屬性說明
Compute Device預設 auto（用全域偏好），可逐運算子覆寫為 CPU 或 CUDA。
Enabled執行開關——可停用某運算子以除錯、調效能或先把整條管線配置好再一起開。
Source指定運算子讀取哪個資料集，通常依你的選取自動設定。

3. 運算子類型#

Faces（外表面）

從體積網格萃取並渲染邊界表面。適合檢視模擬域外型，或以欄位值（壓力、壁面剪應力）為外表面著色。

  • 對多面體資料集，請套用在表面網格區塊而非體積區塊——「從多面體體積萃取內部面尚未支援」。

步驟：右鍵資料集節點 → Create > CAE Operators > Faces → 用欄位選擇以純量資料著色。

Points（點）

在每個網格頂點畫球體，快速概覽資料密度與網格節點分佈；也很適合沒有網格連結性的點雲資料。

步驟：右鍵資料集 → Create > CAE Operators > Points → 調 Width 控制球體大小 → 以欄位著色或依資料值改變尺寸。

Glyphs（字形）

在資料點放置有方向的 3D 形狀，用來視覺化向量欄位。形狀可選箭頭（最適合速度／力）、圓錐（方向指示）、球（僅位置／大小）。

設定：

  • Orientations 旁點 Add Target 選向量欄位分量。
  • Scales 旁 Add Target 改變字形大小。
  • Colors 旁 Add Target 指定著色。

關鍵屬性：

  • Max Count：最多渲染幾個字形（預設 10,000,000；設 0 表示全部）。
  • Use Cell Points：只在網格 cell 的點上放字形。
  • Orientations Mode：eulerAngles（預設）或 quaternion。

Streamlines（串流線）

追蹤粒子在向量欄位中的路徑，揭示流動型態、回流區與尾流結構。需要一個種子來源與一個速度欄位。

步驟：

  1. 先建立一個單位球來源，放到感興趣區域。
  2. 右鍵體積資料集 → Create > CAE Operators > Streamlines。
  3. Seeds 旁 Add Target 加入種子來源。
  4. Velocity 旁 Add Target 加入速度欄位。
  5. 移動球體即可互動式更新串流線。

關鍵屬性：

屬性作用
dx串流線頂點之間的步長。
Max Length最大追蹤距離。
Width顯示寬度。
Region of Interest用邊界框把追蹤限制在空間子集內。
2.0 新增：支援多種子（multi-seed）——一個串流線運算子可同時參考多個種子來源，從不同位置一次追蹤流線做比較分析。

Volume（體積）

用 NVIDIA IndeX 渲染體積資料，兩種路徑：

方法特性
NanoVDB（預設）把資料體素化到規則網格，GPU 渲染，大型資料較快。
Irregular直接在 CPU 渲染非結構化網格，保留 cell 拓撲，需三角或四邊形面。

步驟：右鍵體積資料集 → Create > CAE Operators > Volume → 選渲染方法 → Field 旁 Add Target 選視覺化欄位 →（可選）設 Region of Interest。色彩表在運算子的 Material／Colormap prim 上以互動漸層調整器設定（詳見色彩對應與渲染）。

Slice（切面）

對資料集做平面剖切，檢視內部體積資料。

步驟：

  1. 右鍵資料集 → Create > CAE Operators > Slice。
  2. Field 旁 Add Target 選顯示欄位。
  3. 在 Stage 面板展開運算子，取用 Plane prim。
  4. 用變換 gizmo 移動切面。

可選 Biplane（雙平面）或 Triplane（三平面）做正交剖面。

提示：切面平面的初始位置可能落在可見範圍外——按 F 讓視角貼合（fit）即可找到。

4. 體素化屬性#

套用在 NanoVDB 類運算子（Volume 與 NanoVDB Streamlines）。這些屬性存在運算子 prim 上，確保跨工作階段、跨使用者結果一致：

屬性作用
Max Resolution最長軸上的 cell 數（均勻間距）。
Voxel Size明確指定體素大小（需知道資料 bounds）。
Region of Interest體素化前用邊界框先取子集域。
Explode Bounds把體素化區域擴張到所選 bounds 之外。

5. 欄位分量選擇#

對向量欄位，運算子允許選特定分量或選magnitude（模長）。用 Add Target 從資料集的 FlowSolution 或其他欄位容器選欄位；要多分量選擇（如速度 X、Y、Z），在彈出選單中 Shift-click 各分量。

6. 工作流程訣竅#

  • 用 Stage 面板的眼睛圖示切換運算子可見性，隔離出想看的結果。
  • 用變換 gizmo 選取並移動種子來源（如單位球），互動式探索資料集。
  • 先把整條視覺化管線配置好，再統一開啟 Enabled，方便調效能。
