Kit-CAE 總覽文檔#

對應官方 Overview 與 Kit-CAE 2.0 章節的中文整理。本頁幫你在 10 分鐘內建立 Kit-CAE 的完整心智模型。

1. Kit-CAE 是什麼#

Kit-CAE 是一套用 OpenUSD、NVIDIA CUDA-X 函式庫與 NVIDIA Omniverse 處理工程模擬資料的參考實作（reference implementation）與開發者工具箱。原始碼在 GitHub 公開，想擴充或客製的開發者可自由取用。

它讓你能夠：在 OpenUSD 中匯入、探索並視覺化工程模擬資料。工程資料維持在原生格式（CGNS、HDF5、VTK、EnSight、NumPy、OpenFOAM…），OpenUSD 只擔任「組合層（composition layer）」；資料透過可插拔的 delegate 依需求載入，再由 GPU 加速運算子負責處理與視覺化。

2. 核心理念：不轉檔的組合層#

Kit-CAE 底層的設計原則很單純：

OpenUSD 作為科學資料集的組合與存取層，而原生檔案始終是唯一的權威資料來源（source of truth）。

Kit-CAE 不去轉換工程資料，而是把「真相之源」直接帶進一個可組合的 OpenUSD 舞台。這種做法能容納多種資料來源，並排並存於同一場景：

  • 模擬求解器輸出（CFD、結構、熱傳…）
  • 感測器輸入
  • AI 代理模型（surrogate model）推論結果

對 AI 代理資料而言尤其關鍵——「代理模型的推論資料，在匯入、運算子與視覺化演算法上，都與求解器輸出一視同仁」。

3. 三大架構元件#

實作倚賴三個主要架構元件，你可以直接使用，也可以改造進自有工作流程：

元件角色
CAE 專用 Schema在 USD 上定義工程資料所需的資料模型（資料集、欄位陣列、運算子 API 等），讓 CAE 概念能以 USD prim 表達。
Data Delegate 介面可插拔的資料委派層，負責從原生檔案「依需求」讀取欄位與幾何，避免一次把整份資料灌進記憶體。新格式＝實作新的 delegate。
視覺化運算子（GPU 加速）以 CUDA-X 實作的動態視覺化：外表面、字形、串流線、體積、切面…重載舞台時即時重算。

更深入的 USD schema、Data Delegate API 與運算子規格細節，請參考 GitHub 儲存庫的參考資料。

4. 三種實體：Sources、Operators、Flow#

Kit-CAE 2.0 的介面把場景中的 CAE 元素清楚地分成三類，這是理解整個工具的關鍵：

實體特性例子
Sources（來源）不可變的輔助實體，重載舞台時不會重新執行；是沒有套用 CAE 運算子 API 的普通 USD prim。邊界框、單位球
Operators（運算子）動態視覺化實體，重載舞台時會即時重算，從來源資料即時產生結果。Faces、Streamlines、Volume、Slice、Points、Glyphs
Flow（流場）以 NVIDIA Flow 做的粒子動畫式流場視覺化。煙霧狀流動動畫

5. 主線工作流程#

官方指南以同一份 CFD 汽車空氣力學資料集，用循序漸進的「Try It」段落貫穿主線：

  1. 匯入資料——把原生檔案帶進 USD 舞台
  2. 探索資料——用 CAE Insights 看懂欄位與統計
  3. 建立來源——邊界框（ROI）與單位球（種子）
  4. 建立運算子——外表面、串流線、體積、切面…
  5. 色彩對應與渲染——依欄位著色、色彩表與鎖定
  6. 流場視覺化——NVIDIA Flow 粒子動畫

此外還有四個獨立範例：時變結構分析、模擬資料融入高保真幾何情境、單一場景內多領域模擬、AI 代理模型模擬資料。

6. Kit-CAE 2.0 有什麼新東西#

運算子強化

  • 介面明確區分 Sources／Operators／Flow 三類實體。
  • 逐運算子（per-operator）的裝置選擇、執行開關、指定欄位分量選擇，與屬性鎖定（避免重算覆蓋你的設定）。
  • 多種子（multi-seed）串流線、雙平面／三平面切面，以及可重現的體素化（設定存在運算子 prim 上）。

格式支援

  • 新增直接匯入 OpenFOAM case 目錄（含平行分解案例）。
  • EnSight Gold 支援多面體（polyhedral）元素並快取中繼資料，加速重讀；VTK 讀取器最佳化，匯入更快。

效能

  • 統計計算（邊界框、欄位範圍、直方圖）改為 GPU 加速，大型資料集互動更順。
  • 視覺化 kernel 採 AOT 預編譯並快取，縮短啟動時間。
  • 時間序列匯入提供 Scale／Offset，把模擬步對應到 USD time sample。

易用性

  • CAE Insights：在 Properties 面板直接檢視欄位陣列中繼資料、四分位與直方圖，並與時間軸同步。
  • 影片擷取會等待計算完成才錄製；Flow 體素化有合理預設並跳過已體素化資料。
  • 偏好設定跨工作階段保存；預設改為 Z-up，對齊 CAE 業界慣例。
向後相容：舊版 CAE 選單預設停用，但仍可用於較早建立的舞台，並可在 Preferences 重新啟用。
