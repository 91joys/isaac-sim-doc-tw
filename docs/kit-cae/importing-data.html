匯入資料#

對應官方 Import Data 章節。Kit-CAE 不轉檔——工程資料維持原生格式，匯入只是把它「掛」進 OpenUSD 舞台。

1. 支援的檔案格式#

格式副檔名／型態說明
CGNS.cgns階層式科學資料，完整支援 SIDS 資料模型。最適合來自商用 CFD 求解器的結構化與非結構化網格。
NumPy.npz／.npy來自 Python 工作流程的點雲與陣列資料，適合機器學習管線與快速原型。
VTKVTI／VTU／VTP／VTS 與 legacy結構化與非結構化網格。針對大型 VTU 檔案有讀取最佳化。
EnSight Gold.case網格與欄位資料，支援多面體（polyhedral）元素並快取中繼資料以加速重讀。商用求解器常見輸出格式。
OpenFOAMcase 目錄直接匯入 OpenFOAM case 目錄，含平行分解（decomposed）案例，無需轉檔。
2.0 新增：OpenFOAM 直接匯入為 2.0 新功能；EnSight Gold 的多面體支援與 VTK 讀取器加速也是本版重點。

2. 匯入流程#

  1. 開啟 File > Import，瀏覽到資料目錄。
  2. 確認檔案類型篩選符合你的格式（如 *.cgns、*.npz、*.*）。
  3. 勾選 Import to Stage（建議）把資料直接載入目前舞台。
  4. 點 Import。

匯入後，資料會以新的 prim 出現在 Stage 面板的 /World 底下。

注意：匯入完成後不會立即出現任何視覺化——必須先建立視覺化運算子，畫面上才看得到東西。

3. 時間序列匯入#

對含多個時間步的檔案，先分清兩個概念（以 CGNS 為例）：

概念意義
Time step（時間步）整數索引：0、1、2、3…
Time value（時間值）實際模擬時間，例如 0.001s、0.002s…

匯入時可用兩個對應控制，把模擬步映射到 USD 的 time sample：

控制作用
Scale（縮放）乘以時間步索引。Scale 設 10 時，step 1 會落在 sample 10——為兩步之間預留插補影格。
Offset（偏移）把所有 time sample 平移一個常數。

這些設定讓時間步之間能平滑插補，供動畫使用。

4. 多檔匯入#

可一次選取多個檔案，並為每個檔案設定各自的匯入選項。

目前限制：尚無法「選取多檔並一次套用相同選項」——每個檔案的選項需個別設定。

5. 單位處理#

  • Kit-CAE 以 公分（cm）為預設單位。
  • 匯入時不做任何自動單位換算。
  • 若你的資料使用不同單位，需自行以變換（transform）調整。

6. 試試看：匯入汽車空氣力學資料#

  1. File > Import，選擇 auto_aero_solver_result.cgns。
  2. 確認篩選為 *.cgns，勾選 Import to Stage。
  3. 點 Import，等待約 790 萬節點的資料掛入舞台。
  4. 到 Stage 面板 /World 下確認新 prim 已出現。
  5. 接著前往探索資料，用 CAE Insights 檢視欄位。
