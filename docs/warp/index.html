建議學習路徑

  1. 總覽文檔：先理解 Warp 的定位——kernel 級 GPU 程式設計，而非張量框架。
  2. 安裝與快速入門：pip install warp-lang，跑起第一個 GPU kernel。
  3. 基礎入門：掌握 @wp.kernel → wp.launch() → wp.array 的資料流與編譯模型。
  4. 執行期核心功能：陣列進階操作與 Mesh／Volume／HashGrid 空間計算原語。
  5. 可微分性：用 wp.Tape() 讓梯度反向流過整段模擬。
  6. 互操作性：把 Warp kernel 接進 PyTorch／JAX 訓練管線。
  7. 依方向深入：Tile 程式設計（效能）、多 GPU 與 CUDA Graph（規模化）、領域模組（FEM／稀疏矩陣），或到 FAQ 查限制與除錯技巧。
  8. 做機器人模擬／強化學習：閱讀 MuJoCo Warp 總覽與使用，再依需求進到效能調校與差異與 FAQ。
