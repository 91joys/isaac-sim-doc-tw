安裝與部署#

整理自官方 Installation 與 Deployment 章節。安裝 Isaac Lab 前必須先安裝 Isaac Sim——Isaac Lab 建構於其上。

系統需求#

項目需求
作業系統Ubuntu 22.04 或 Windows 11
記憶體至少 32 GB RAM
GPUNVIDIA RTX 系列，至少 16 GB VRAM
Python需與 Isaac Sim 綁定版本一致（Isaac Sim 5.X 為 Python 3.11）
驅動程式Linux 建議 NVIDIA driver 580.65.06 以上

版本相容性：Isaac Sim 是針對特定 Python 版本編譯的，因此 Isaac Lab 使用的 Python 必須與之相符。Isaac Sim 4.2.0 以下版本已停止支援。

Pip 安裝（推薦流程）#

以下為官方推薦的最快路徑，以 conda／venv 建立獨立環境後，用 pip 安裝 Isaac Sim，再由原始碼安裝 Isaac Lab。

1. 建立虛擬環境#

# 以 conda 建立 Python 3.11 環境（版本需與 Isaac Sim 相符）
conda create -n env_isaaclab python=3.11
conda activate env_isaaclab

# 升級 pip
pip install --upgrade pip

2. 以 pip 安裝 Isaac Sim#

# 先安裝對應 CUDA 版本的 PyTorch，再安裝 Isaac Sim 套件組
pip install torch --index-url https://download.pytorch.org/whl/cu128
pip install "isaacsim[all,extscache]" --extra-index-url https://pypi.nvidia.com

3. 取得並安裝 Isaac Lab#

# 複製原始碼
git clone https://github.com/isaac-sim/IsaacLab.git
cd IsaacLab

# 安裝所有 Isaac Lab 擴充與相依（含 RL 框架）
./isaaclab.sh --install        # Windows：isaaclab.bat --install

若只想安裝特定 RL 框架相依，可用 ./isaaclab.sh --install rsl_rl（可替換為 rl_games、skrl、sb3）。

4. 驗證安裝#

# 開一個空的模擬視窗
./isaaclab.sh -p scripts/tutorials/00_sim/create_empty.py

# 跑一個內建訓練（headless、少量環境快速驗證）
./isaaclab.sh -p scripts/reinforcement_learning/rsl_rl/train.py \
    --task Isaac-Cartpole-v0 --headless --num_envs 64

isaaclab.sh 常用指令#

isaaclab.sh（Windows 為 isaaclab.bat）是與 Isaac Lab 互動的主要入口，會自動使用 Isaac Sim 綁定的 Python 直譯器。

指令用途
-i / --install [框架]安裝擴充與相依（可指定 RL 框架）
-p / --python <檔案>以 Isaac Sim 的 Python 執行腳本
-s / --sim啟動 Isaac Sim 模擬器
-f / --format對程式碼執行格式化
-t / --test執行測試
-v / --vscode產生 VS Code 設定
-h / --help顯示說明

容器與雲端部署#

原文連結

  • Docker：官方提供 docker/ 目錄與腳本，封裝 Isaac Sim ＋ Isaac Lab，適合可重現的訓練環境與 CI。
  • 叢集（Cluster）：支援在 SLURM／PBS 等 HPC 叢集上以容器提交批次訓練。
  • CloudXR：串流互動式渲染以便遠端檢視。
  • 雲端部署：可在 AWS、GCP、Azure、阿里雲等雲端 GPU 執行個體部署，搭配 Ray 進行分散式訓練與超參數搜尋。

資產快取#

Isaac Lab 的機器人與場景資產以 USD 形式存於 NVIDIA 的 Nucleus／雲端伺服器。為加速載入並支援離線工作流程，可啟用資產快取（Asset Caching）：快取後的資產儲存在本機，減少重複下載，對頻寬受限的環境尤其有用。
