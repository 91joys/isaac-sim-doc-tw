建議學習路徑

  1. 認識 Isaac Lab：先弄清楚 Isaac Lab 與 Isaac Sim、Isaac Gym 的關係與定位。
  2. 安裝與部署：選定安裝方式（新手建議 pip），把 Isaac Lab 跑起來並驗證安裝。
  3. 核心概念與架構：建立 App／Scene／Environment／Manager 的心智模型，理解兩種工作流程的取捨。
  4. 教學系列：從空場景與剛體／關節互動開始，逐步走到建立 Manager-Based／Direct RL 環境並註冊。
  5. 挑選環境＋訓練策略：從 Isaac-Cartpole-v0 等入門任務起步，用 RSL-RL 等框架跑第一個訓練。
