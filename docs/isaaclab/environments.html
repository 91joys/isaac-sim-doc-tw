可用環境#

整理自官方 Available Environments 章節。Isaac Lab 內建大量現成任務，涵蓋單智能體與多智能體，皆以 gym.make("Isaac-...-v0") 建立。

任務命名慣例#

任務 ID 遵循以下模式：

Isaac-[任務]-[機器人]-[變體]-[工作流程]-v[版本]
範例 ID解讀
Isaac-Lift-Cube-Franka-v0Franka 手臂抬升方塊，Manager-Based（無 -Direct 後綴）
Isaac-Velocity-Rough-Anymal-C-Direct-v0ANYmal-C 在崎嶇地形速度追蹤，Direct 工作流程
Isaac-Cartpole-RGB-Camera-Direct-v0Cartpole 加 RGB 攝影機變體，Direct 工作流程
  • 後綴 -Direct 表示 Direct 工作流程；無此後綴多為 Manager-Based。
  • 版本號自 v0 起算。
  • 可用 scripts/environments/list_envs.py 列出當前所有已註冊環境。
./isaaclab.sh -p scripts/environments/list_envs.py

類別說明代表任務

多智能體任務#

Isaac Lab 也提供支援同時控制多個智能體的環境，涵蓋經典控制與操作任務，可搭配 SKRL 的多智能體演算法（MAPPO、IPPO）訓練。

快速上手一個環境#

先用零動作或隨機動作腳本確認環境可正常運行，再進入訓練：

# 以隨機動作跑一個環境（確認可正常步進與渲染）
./isaaclab.sh -p scripts/environments/random_agent.py \
    --task Isaac-Cartpole-v0 --num_envs 32

# 以零動作跑（觀察被動動力學）
./isaaclab.sh -p scripts/environments/zero_agent.py \
    --task Isaac-Cartpole-v0 --num_envs 32

選定環境後，接著到強化學習頁挑選 RL 框架開始訓練。入門建議從 Isaac-Cartpole-v0 或 Isaac-Ant-v0 這類低維度任務起步。
