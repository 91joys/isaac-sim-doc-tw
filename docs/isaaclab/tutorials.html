教學系列#

整理自官方 Tutorials 章節，依六大主題循序漸進。每個教學都對應 scripts/tutorials/ 下的可執行腳本，建議搭配官方原始碼實作。

教學執行方式一致——以 isaaclab.sh -p 執行對應腳本，例如：

./isaaclab.sh -p scripts/tutorials/00_sim/create_empty.py

一、設定簡單模擬#

scripts/tutorials/00_sim/

教學學到什麼
建立空場景（Creating an empty scene）啟動最基本的模擬：AppLauncher、SimulationContext、步進迴圈
生成 Prim 到場景（Spawning prims）用 spawn 設定加入地面、光源、幾何物件與 USD 資產
深入 AppLauncher（Deep-dive into AppLauncher）掌握 AppLauncher API 與命令列參數（headless、裝置、攝影機）

二、與資產互動#

scripts/tutorials/01_assets/

教學學到什麼
加入新機器人（Adding a New Robot）把自訂機器人（URDF／USD）整合進 Isaac Lab 的資產設定
與剛體互動（Rigid object）讀寫剛體的位姿與速度，施加外力
與關節體互動（Articulation）控制多關節機構：設定關節目標、讀取關節狀態
與可變形體互動（Deformable object）操作柔性／軟體材質物件
與表面夾爪互動（Surface gripper）夾爪機構與表面接觸的處理

三、建立場景#

scripts/tutorials/02_scene/

教學學到什麼
使用 Interactive Scene以 InteractiveScene 高階抽象一次宣告地形、機器人與感測器，並自動複製成多個平行環境

四、設計環境#

scripts/tutorials/03_envs/——本主題是 RL 的核心，從建立環境到訓練代理。

教學學到什麼
建立 Manager-Based Base 環境用設定類別組裝觀測、動作等 Manager，建立非 RL 的基礎環境
建立 Manager-Based RL 環境加入獎勵、終止、事件 Manager，完成可訓練的 RL 環境
建立 Direct 工作流程 RL 環境不經 Manager，直接實作觀測／獎勵／終止方法
註冊環境（Registering）用 gymnasium.register() 給環境 Isaac-...-v0 的 ID
以 RL 代理訓練（Training with an RL Agent）串接 RL 框架跑起第一個訓練
設定 RL 代理（Configuring an RL Agent）管理超參數與框架設定
修改既有 Direct RL 環境在現成環境上客製化
USD 環境中的策略推論載入已訓練策略在 USD 場景中執行

五、整合感測器#

scripts/tutorials/04_sensors/

教學學到什麼
在機器人上加感測器把攝影機、接觸感測器、IMU、ray caster 等掛到機器人並讀取輸出

六、使用運動生成器#

scripts/tutorials/05_controllers/

教學學到什麼
使用任務空間控制器（Task-space controller）差分逆運動學（Differential IK）：在末端執行器空間下指令
使用操作空間控制器（Operational Space Controller）OSC：在任務空間控制力／阻抗，適合接觸密集與柔順任務

延伸：How-to 指南#

官方另有 How-to Guides 針對特定操作提供實用範例：

  • 匯入新資產（URDF／MJCF／Mesh 轉 USD）。
  • 撰寫環境設定與 CLI 參數處理。
  • 渲染與錄影、除錯視覺標記（Markers）。
  • 效能調校與大規模訓練最佳化。

建議順序：先跑完主題一～三建立基礎，再進入主題四完成第一個可訓練環境，最後依需求補上感測器與控制器。訓練框架與指令見強化學習頁。
