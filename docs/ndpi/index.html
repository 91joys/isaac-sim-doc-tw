建議學習路徑

  1. 編譯與快速上手：把 nDPI 編起來，拿自己的 PCAP 跑 ndpiReader -v 2 看輸出。
  2. 核心架構＋偵測管線：建立「三大結構、五層策略」的心智模型。
  3. Flow 追蹤＋協定堆疊＋風險評估：理解偵測結果怎麼表達、狀態怎麼維護。
  4. 整合：照骨架程式碼把 nDPI 嵌進自己的程式。
  5. 進階：協定解析器 → 資料結構 → 擴充新協定（寫一個自己的 dissector）。
