風險評估系統（Flow Risk）#

對應 DeepWiki 2.4 Risk Assessment and Security Analysis 與官方 doc/flow_risks.rst。nDPI 在辨識協定的同時，標記 57 種安全與政策風險，存在每條 flow 的 64-bit 位圖中。

風險的組成#

每種風險在 ndpi_known_risks[] 中定義四項屬性：

typedef struct {
  ndpi_risk_enum risk;                      /* 風險識別碼（0-56）*/
  ndpi_risk_severity severity;              /* LOW/MEDIUM/HIGH/SEVERE/CRITICAL/EMERGENCY */
  risk_percentage default_client_risk_pctg; /* 用戶端責任比例（0-100%）*/
  ndpi_risk_accountability accountability;  /* CLIENT / SERVER / BOTH */
} ndpi_risk_info;

Flow 端儲存三樣東西：flow->risk（64-bit 位圖，一 bit 一風險）、flow->risk_score（加權分數）、flow->risk_infos[]（最多 8 則說明訊息）。

嚴重度與計分#

嚴重度分數意義
LOW10次要問題（用 IP 直連、明文協定）
MEDIUM50中度疑慮（非標準 port 跑已知協定）
HIGH100顯著問題（弱加密、自簽憑證、DGA 網域）
SEVERE150嚴重威脅（XSS、SQL injection、接觸惡意主機）
CRITICAL / EMERGENCY200 / 250保留未來使用

總分算法：走訪 flow->risk 每個被設起的 bit → 取其嚴重度分數 → 依 client % 拆成用戶端與伺服器端兩份 → 各自加總。責任歸屬（accountability）標示是誰觸發：XSS 是 client、過期憑證是 server、不安全協定是雙方。

常見風險範例#

風險嚴重度觸發情境偵測位置
NDPI_URL_POSSIBLE_XSSSEVEREURL 內含 script 注入（libinjection 判定）HTTP 解析器
NDPI_URL_POSSIBLE_SQL_INJECTIONSEVEREURL 內含 SQL 注入樣式HTTP 解析器
NDPI_MALWARE_HOST_CONTACTEDSEVERE連到已知惡意主機IP／網域清單
NDPI_TLS_SELFSIGNED_CERTIFICATEHIGHTLS 憑證自簽（責任在 server）TLS 解析器
NDPI_MALICIOUS_FINGERPRINTHIGHJA4 等指紋命中已知惡意特徵指紋雜湊表
NDPI_SUSPICIOUS_DGA_DOMAINHIGH網域疑似演算法產生（DGA）DNS／TLS 解析器

各協定的風險檢查點#

協定檢查項目
TLS / DTLS自簽／過期／即將過期／效期過長憑證、惡意 SHA1 指紋、過舊版本（< TLS 1.2）、弱加密套件、缺 SNI、ALPN 與 SNI 不一致
HTTPURL 注入（XSS / SQL / RCE）、可疑 User-Agent、爬蟲、二進位執行檔傳輸、可疑 Content-Type、過舊伺服器版本、可疑標頭
DNSDGA 網域、Punycode IDN、非法字元、大封包（UDP > 512 bytes 無 EDNS）、分片、可疑紀錄型別、過長首標籤（疑似資料外洩）
SSH過舊的用戶端／伺服器版本或弱演算法

在程式中使用風險#

/* 解析器內標記風險 */
ndpi_set_risk(ndpi_struct, flow, NDPI_TLS_WEAK_CIPHER, "Weak cipher detected");

/* 檢查特定風險 */
if (NDPI_ISSET_BIT(flow->risk, NDPI_MALWARE_HOST_CONTACTED)) {
    /* 處理惡意主機事件 */
}

/* 走訪全部風險 */
for (int i = 0; i < NDPI_MAX_RISK; i++)
    if (NDPI_ISSET_BIT(flow->risk, i))
        printf("%s\n", ndpi_risk2str(i));

/* 依分數過濾高風險流量 */
if (flow->risk_score > threshold) { /* 告警 */ }

風險例外（抑制誤報）#

可依主機名稱樣式、IP 範圍、憑證 DN 設定例外規則；風險被標記後會經過 ndpi_check_flow_risk_exceptions()，命中例外即清掉該 bit——例如內部 CA 的自簽憑證不再告警。各協定的風險偵測也能獨立調整（停用特定檢查、調整門檻）。

Wireshark 整合#

Wireshark 外掛（wireshark/ndpi.lua）把 nDPI 的 64-bit 風險位圖展開成 57 個布林欄位與分數欄位，可直接用顯示過濾器：

ndpi.flow_risk.malware_host_contacted
ndpi.flow_risk.tls_selfsigned_certificate
ndpi.flow_score > 100
