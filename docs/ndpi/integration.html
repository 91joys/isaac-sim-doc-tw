整合到你的應用程式#

對應 DeepWiki 5.1 ndpiReader Application 與 6. External Integrations。本頁提供整合骨架程式碼，細節請對照 example/ndpiReader.c 與 example/reader_util.c。

整合骨架程式碼#

/* ① 初始化（程式啟動時做一次）*/
struct ndpi_detection_module_struct *ndpi_str = ndpi_init_detection_module(NULL);
ndpi_set_config(ndpi_str, NULL, "...", "...");   /* 選用：載入自訂規則 */
ndpi_finalize_initialization(ndpi_str);          /* 建好比對結構 */

/* ② 每條新連線：配置 flow 狀態（flow 表由「你的程式」管理）*/
struct ndpi_flow_struct *flow = ndpi_flow_malloc(SIZEOF_FLOW_STRUCT);
memset(flow, 0, SIZEOF_FLOW_STRUCT);

/* ③ 每個封包：交給 nDPI，直到分類完成 */
ndpi_protocol res = ndpi_detection_process_packet(
    ndpi_str, flow, ip_packet, ip_len, time_ms, &input_info);

if (res.state == NDPI_STATE_CLASSIFIED) {
    /* 分類完成：可讀取最終協定與所有 metadata */
    char buf[64];
    printf("%s\n", ndpi_protocol2name(ndpi_str, res, buf, sizeof(buf)));
} else {
    /* nDPI 還需要更多封包；
       NDPI_STATE_PARTIAL 時 res.proto 是可用的初步結果 */
}

/* ④ flow 逾時仍未分類：放棄偵測，做最後猜測 */
ndpi_protocol final = ndpi_detection_giveup(ndpi_str, flow);

/* ⑤ 回收 flow */
ndpi_free_flow_data(flow);
ndpi_flow_free(flow);

/* ⑥ 程式結束 */
ndpi_exit_detection_module(ndpi_str);

整合關鍵點#

整合點函式／結構備註
模組初始化ndpi_init_detection_module()建立偵測情境（一次）
封包處理ndpi_detection_process_packet()核心偵測 API（逐包）
Flow 管理ndpi_flow_struct由應用程式建立／查表／回收
設定ndpi_set_config()協定專屬調校
取結果ndpi_get_flow_*() 系列萃取偵測結果與 metadata

執行緒安全：偵測模組 finalize 後為唯讀、可跨執行緒共享；flow 結構不可在無同步下共用。記憶體：自訂配置器要在任何 nDPI API 使用之前安裝。

ndpiReader 的架構借鏡#

  • 多執行緒：每個 reader thread 有獨立的 ndpi_workflow、pcap handle、flow 表與統計；主執行緒最後彙整。
  • Flow 表：uthash 雜湊表 + 多個獨立 root（NUM_ROOTS）分散競爭。
  • 閒置掃描：定期走訪 flow 表，逾時的 flow 先 ndpi_detection_giveup() 再輸出與回收。
  • 工作流函式：ndpi_workflow_init() → ndpi_workflow_process_packet()（查表／建 flow ＋ 呼叫 nDPI）→ ndpi_idle_scan_walker() → ndpi_workflow_free()。

輸出格式#

ndpiReader 支援 console、CSV、JSON（ndpi_serializer，另支援 TLV）、流量指紋輸出，以及 Wireshark extcap 整合（把協定與風險寫進封包 trailer，配合 ndpi.lua 過濾）。ndpi_serializer 也可直接在你的程式中使用，序列化協定、風險與 metadata。

語言綁定與生態系#

專案說明
Python binding官方 python/ 目錄提供綁定
nfstream基於 nDPI 的 Python 流量分析框架
nDPId輕量 daemon，把 nDPI 結果以 JSON 事件送出
ntopng / nProbe centontop 官方產品，用 nDPI 做流量辨識與選擇性封鎖
Wireshark 外掛wireshark/ndpi.lua：協定與 57 種風險變成可過濾欄位

用 nDPI 封鎖流量？#

nDPI 本身只是函式庫，不做封鎖；把它嵌進你的網路應用（inline 位置）即可依分類結果選擇性阻擋。官方範例是 ntopng 與 nProbe cento。
