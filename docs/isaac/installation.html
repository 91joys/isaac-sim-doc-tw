安裝與設定（Installation）#

對應官方 Installation 章節的中文導覽。快速安裝步驟與系統需求詳表見《完整文檔》第 2、3 章，本頁聚焦各安裝方式的差異、選擇建議與完整頁面對照。

怎麼選安裝方式#

你的情境 建議方式
第一次接觸、想快速看到效果 Quick Install（standalone 壓縮包，解壓即用）
日常桌面開發 Workstation 安裝（完整應用＋本機相依）
需要可重現環境、CI、叢集 容器（Docker）——僅限 Linux
Python 優先、想用 pip/conda 管理 Python 環境安裝（pip install isaacsim）
本機沒有夠力的 GPU 雲端部署（AWS／Azure／GCP／Brev 等）＋ WebRTC 串流
搭配 ROS 2 開發 任一方式＋ ROS 2 安裝

安裝前建議先跑 Isaac Sim Compatibility Checker（輕量相容性檢查工具，見 Workstation 安裝頁）確認硬體符合需求。

各安裝方式重點#

Quick Install（快速安裝）#

官方頁面。下載 standalone 壓縮包 → 解壓 → 執行 post_install → 啟動 isaac-sim.sh/.bat。注意：保留 50 GB 磁碟空間、首次啟動需數分鐘暖機、資產需 HTTPS 連線。完整指令見《完整文檔》3.1 節。

Workstation 安裝#

官方頁面。完整桌面應用安裝流程，含相容性檢查工具、選用元件與桌面捷徑設定。

容器安裝（Container）#

官方頁面。從 NGC 拉取官方映像在 Docker 中執行，僅支援 Linux，需要 NVIDIA Container Toolkit。適合無頭伺服器、叢集與 CI；搭配串流用戶端可遠端看畫面。

Python 環境安裝（pip / conda）#

官方頁面｜PyPI。把 Isaac Sim 當 Python 套件裝進虛擬環境，之後直接用該環境的 python 跑 standalone 腳本——Python 優先工作流程與 Isaac Lab 使用者的首選。

雲端部署#

總覽頁面。官方提供各雲平台部署指南：

  • Isaac Launchable／NVIDIA Brev（最快上雲）
  • AWS｜Azure｜Google Cloud
  • 騰訊雲｜阿里雲｜火山引擎｜百度雲
  • 遠端工作站

串流用戶端（Livestream Clients）#

官方頁面。遠端／無頭機器上以 isaac-sim.streaming.sh/.bat 啟動，本機用 Isaac Sim WebRTC Streaming Client 連線觀看與操作。

ROS 2 安裝#

預設平台｜其他平台。設定 ROS 2 Bridge 所需的 ROS 環境；細節與教學見本站《ROS 2》章節頁。

下載與疑難排解#

  • 下載 Isaac Sim——各平台最新版與歷史版本。
  • Setup Tips（安裝技巧與 FAQ）——常見安裝問題、本地資產包（離線環境必看）、快取設定。
  • Troubleshooting——啟動失敗、驅動、渲染問題總表。
