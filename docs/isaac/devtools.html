OmniGraph（視覺化編程）#

節點圖框架，GUI 端機器人控制器與 ROS 2 Bridge 的底層。學習路徑：

  1. OmniGraph 核心概念與介面（Omniverse 文檔）。
  2. Isaac Sim OmniGraph 教學——建立差速機器人控制圖。
  3. 以 Python 腳本操作 OmniGraph——程式化建圖與設值。
  4. 自訂節點：Python 節點｜C++ 節點｜IPC 節點。
  5. 常用捷徑——Tools > Robotics > OmniGraph Controllers 系列快速產生器。

瀏覽器（Browsers）#

總覽：Content Browser（檔案與資產）、Material Browser、NVIDIA Asset Browser、SimReady Explorer（帶物理屬性的即用資產）。

開發流程建議#

  1. 探索期：Script Editor＋Commands Tool（看 GUI 操作對應什麼 API）。
  2. 迭代期：Extension＋熱重載，或 VS Code＋Python Server。
  3. 量產期：Standalone 腳本（無頭、可 CI），需要時用 Tracy 剖析效能。
