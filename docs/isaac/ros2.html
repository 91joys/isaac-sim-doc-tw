ROS 2 整合（ROS 2）#

對應官方 ROS 2 章節的中文導覽。Isaac Sim 透過 ROS 2 Bridge 讓模擬世界成為 ROS 2 生態的一個節點：發布感測器資料、訂閱控制命令、以服務控制模擬本身。

核心概念#

  • ROS 2 Bridge 以 OmniGraph 節點實作：相機發布器、TF 發布器、Twist 訂閱器等都是圖上的節點，用 GUI 或 Python 建圖即可接上 ROS 2。
  • 支援 Linux 與 Windows；先完成 ROS 2 安裝（預設）或其他平台版本。
  • Standalone 工作流程也能用 bridge（ROS 2 Bridge in Standalone Workflow），可精準控制發布時序。
  • 架構建議見 ROS 2 Reference Architecture。

建議學習路徑#

  1. TurtleBot 系列（入門主線）：URDF 匯入 TurtleBot → 用 ROS 2 訊息駕駛 → Clock 時鐘同步 → TF 樹與里程計。
  2. 感測器發布：相機（含發布細節、加噪聲、壓縮影像）、RTX 光達、RTX 雷達。
  3. 品質與時序：發布頻率設定、QoS、RTF 即時係數發布。
  4. 導航與操作：Nav2 導航、多機器人導航、高度圖導航、MoveIt 2、關節控制（Extension Python）、Ackermann 控制器。
  5. 進階整合：以 ROS 2 服務控制模擬（載入世界、生成實體、步進）、透過 ROS 2 執行 RL 策略、綜合應用 Putting It All Together。
