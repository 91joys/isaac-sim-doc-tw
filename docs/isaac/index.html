建議學習路徑

  1. 安裝與設定：選擇安裝方式，把 Isaac Sim 跑起來（詳細指令見總覽文檔第 3 章）。
  2. 快速入門＋GUI 介面：用滑鼠完成第一個會掉落的方塊與 Franka 手臂。
  3. 核心概念：建立三大工作流程與 World／Scene／Stage 的心智模型。
  4. 實戰教學三部曲：Script Editor（互動探索）→ Extension（UI 工具開發）→ Standalone Python（無頭批次）。
  5. 依方向深入專題：ROS 2、合成資料、Isaac Lab、機器人設定、IMU 感測器教學（AMR 運動驗證）、AMR 導航 SDG 教學（Replicator 實戰）。
