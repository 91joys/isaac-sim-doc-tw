## 官方資產庫（Assets）

官方雲端資產庫提供大量即用 USD 資產，程式內以 `get_assets_root_path()` 取得根路徑；GUI 內從 **Create** 選單或 Asset Browser 取用。離線環境請安裝[本地資產包](https://docs.isaacsim.omniverse.nvidia.com/latest/installation/install_faq.html)。

| 類別 | 內容 | 連結 |
|---|---|---|
| 機器人資產 | Franka、UR 系列、Nova Carter、四足、人形等數十款 | [清單](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_robots.html) |
| 感測器資產 | [相機／深度相機](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_camera_depth_sensors.html)（RealSense 等）、[非視覺感測器](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_nonvisual_sensors.html)（各廠牌光達） | [總覽](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_sensors.html) |
| 道具資產 | 箱子、棧板、工具等 SimReady 物件 | [清單](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_props.html) |
| 環境資產 | 倉庫、辦公室、醫院、Simple Room 等 | [清單](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_environments.html) |
| 精選資產 | [Nova Carter](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/nova_carter_landing_page.html) 等專頁 | [清單](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_featured.html) |
| 第三方資產 | 社群與夥伴提供的 USD 資產 | [清單](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_third_party.html) |
| NuRec 神經渲染 | Gaussian-splat 重建場景的載入與[渲染工具](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/nurec_utils.html) | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_nurec.html) |

## Omniverse 與 USD

USD（OpenUSD）是 Isaac Sim 的場景基石，深入開發前建議理解：

| 主題 | 內容 | 連結 |
|---|---|---|
| OpenUSD 基礎 | prim、屬性、組合（composition）、layer 等核心概念 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/open_usd.html) |
| 操作 USD | 以 Python（`pxr`）讀寫 stage 的實務 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/intro_to_usd.html) |
| USD 工具 | usdview 等命令列工具 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/usd_tools.html) |
| Robot Schema | 機器人在 USD 中的標準結構規範 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/robot_schema.html) |
| Sensor Schema | 感測器的 USD 規範 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/sensor_schema.html) |
| Commands | Omniverse 命令系統（搭配 Commands Tool 使用） | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/omniverse_tools.html) |
| C++ USD 外掛 | 對 standalone 安裝建置 C++ USD 外掛 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/omniverse_usd/building_cpp_usd_plugins.html) |
