# 機器人：匯入、設定與模擬（Robots）

> 對應官方 [Importers and Exporters](https://docs.isaacsim.omniverse.nvidia.com/latest/importer_exporter/importers_exporters.html)、[Robot Setup](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/index.html)、[Robot Simulation](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/index.html) 與 [Robot Motion (Experimental)](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_motion_experimental/index.html) 章節的中文導覽。這是把「一台機器人」從描述檔帶進模擬、調到能用、控到會動的完整旅程。

---

## 整體流程

```
描述檔 (URDF / MJCF / CAD / USD)
   → 匯入（Importers）
   → 設定與調校（Robot Setup：關節、碰撞、增益）
   → 模擬與控制（Articulation、控制器、夾爪）
   → 進階運動規劃（cuMotion / PINK，實驗性）
```

## 2. 機器人設定（Robot Setup）

**檢查與編輯工具**：

- [Robot Inspector](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/robot_inspector.html)——檢視機器人結構與屬性。
- [Robot Poser](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/robot_poser.html)——擺姿勢、存姿態。
- [Robot Assembler](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/assemble_robots.html)——把手臂＋夾爪、車＋手臂組裝成一台。
- [Gain Tuner](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/ext_isaacsim_robot_setup_gain_tuner.html)——關節增益（stiffness/damping）調校。
- [自碰撞偵測](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/ext_isaacsim_robot_setup_collision_detector.html)、[網格合併工具](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/ext_isaacsim_util_merge_mesh.html)。
- [Joint Inspector](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_setup/joint_inspector.html)／[Physics Inspector](https://docs.isaacsim.omniverse.nvidia.com/latest/physics/joint_inspector.html)——關節極限、驅動參數即時檢視。

## 3. 機器人模擬與控制（Robot Simulation）

- [Articulation Controller](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/articulation_controller.html)——關節層控制核心（Python 用法見《完整文檔》7.3 節與《Script Editor 教學》第 6 課）。
- [移動機器人控制器](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/mobile_robot_controllers.html)——差速、Ackermann 等。
- [Surface Gripper（吸附式夾爪）](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/ext_isaacsim_robot_surface_gripper.html)、[Grasp Editor](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/grasp_editor.html)。
- [RL 策略範例](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/ext_isaacsim_robot_policy_example.html)——在 Isaac Sim 內執行訓練好的策略（四足、人形等）。
- [模擬技巧](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/robot_simulation_tips.html)與[核心概念連結集](https://docs.isaacsim.omniverse.nvidia.com/latest/robot_simulation/robot_simulation_core_concepts.html)。
- [Newton Actuators 教學](https://docs.isaacsim.omniverse.nvidia.com/latest/newton_actuators_tutorials/index.html)——Newton 引擎的致動器：[Python 設定](https://docs.isaacsim.omniverse.nvidia.com/latest/newton_actuators_tutorials/newton_actuators_python.html)、[USD 撰寫](https://docs.isaacsim.omniverse.nvidia.com/latest/newton_actuators_tutorials/newton_actuators_usd.html)、[OmniGraph 驅動](https://docs.isaacsim.omniverse.nvidia.com/latest/newton_actuators_tutorials/newton_actuators_omnigraph.html)。

**已棄用但仍在文檔中的 Motion Generation（Lula）**：[總覽](https://docs.isaacsim.omniverse.nvidia.com/latest/manipulators/motion_generation_overview.html)——RMPflow、RRT、運動學求解器、軌跡生成等，新專案請改用下方實驗性方案。
