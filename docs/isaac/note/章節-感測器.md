# 感測器（Sensors）

> 對應官方 [Sensors](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/index.html) 章節的中文導覽。Isaac Sim 的感測器分三大家族：**相機**、**RTX 感測器**（光線追蹤模擬）、**物理式感測器**（物理引擎資料），各有適用場景。

---

## 怎麼選感測器

| 需求 | 家族 | 原理 |
|---|---|---|
| RGB／深度影像、視覺感知 | 相機感測器 | 渲染管線 |
| 光達／雷達點雲（高擬真、材質反射） | RTX 感測器 | RTX 光線追蹤，支援非視覺材質屬性 |
| IMU、接觸力、關節力矩 | 物理式感測器 | 直接讀物理引擎狀態 |
| 簡單射線量測（高效能、低擬真） | 物理射線／PhysX 感測器 | 物理場景查詢 |

> **6.0 重要變更**：相機、RTX 與物理感測器的 API 皆已改版為 Experimental API。從 4.x/5.x 升級請看[遷移指南](https://docs.isaacsim.omniverse.nvidia.com/latest/migration_guides/isaac_sim_6_0/index.html)（含相機、RTX、物理感測器、PhysX 光達等逐項對照）。

## 相機感測器（Camera）

[總覽](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_camera.html)：

- RGB 相機：焦距、光圈、裁切等以 USD 相機屬性設定（程式取得相機參數的片段見《完整文檔》6.4 節）。
- [深度感測器](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_camera_depth.html)——立體深度相機模擬。
- [結構光相機](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_camera_structured_light.html)——投射圖案的主動式深度。

現成相機資產（RealSense 等品牌型號）見 [Camera and Depth Sensor Assets](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_camera_depth_sensors.html)。

## RTX 感測器

[總覽](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx.html)——用 RTX 光線追蹤模擬主動式感測器，可考慮材質反射率等非視覺屬性：

| 感測器 | 連結 |
|---|---|
| RTX Lidar（光達） | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx_lidar.html) |
| RTX Radar（雷達） | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx_radar.html) |
| RTX Acoustic（聲學） | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx_acoustic.html) |

配套主題：

- [RTX Sensor Annotators](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx_annotators.html)——取得點雲與標註輸出。
- [非視覺材質](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx_materials.html)——設定材質對光達／雷達的反射特性。
- [Multi-Tick Rendering](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_multitick_rendering.html)——一個渲染 frame 內多次感測器取樣。
- [自訂 RTX 感測器 Profile](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_rtx_custom.html)——描述市售型號的掃描模式與參數。

現成光達等資產見 [Non-Visual Sensor Assets](https://docs.isaacsim.omniverse.nvidia.com/latest/assets/usd_assets_nonvisual_sensors.html)。

## 物理式感測器（Physics-based）

[總覽](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics.html)——從物理引擎直接取資料，無需渲染：

| 感測器 | 量測 | 連結 |
|---|---|---|
| IMU | 加速度、角速度、姿態 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics_imu.html) |
| Contact（接觸） | 接觸力、接觸事件 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics_contact.html) |
| Articulation joint（關節） | 關節內力／力矩 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics_articulation_force.html) |
| Effort | 關節驅動力 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics_effort.html) |
| Joint state | 關節位置／速度 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics_joint_state.html) |
| Physics raycast | 射線距離量測 | [說明](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physics_raycast.html) |

## PhysX SDK 感測器

[總覽](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physx.html)——基於 PhysX 場景查詢的輕量感測器：[泛用感測器](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physx_generic.html)、[PhysX 光達](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physx_lidar.html)、[光束感測器](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physx_lightbeam.html)、[近接感測器](https://docs.isaacsim.omniverse.nvidia.com/latest/sensors/isaacsim_sensors_physx_proximity.html)。

> 注意：6.0 起 PhysX 光達／泛用／光束感測器已被 **Physics raycast 感測器**取代（見遷移指南），新專案優先使用後者。
