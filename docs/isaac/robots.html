機器人：匯入、設定與模擬（Robots）#

對應官方 Importers and Exporters、Robot Setup、Robot Simulation 與 Robot Motion (Experimental) 章節的中文導覽。這是把「一台機器人」從描述檔帶進模擬、調到能用、控到會動的完整旅程。

整體流程#

描述檔 (URDF / MJCF / CAD / USD)
   → 匯入（Importers）
   → 設定與調校（Robot Setup：關節、碰撞、增益）
   → 模擬與控制（Articulation、控制器、夾爪）
   → 進階運動規劃（cuMotion / PINK，實驗性）

1. 匯入與匯出（Importers & Exporters）#

工具 用途 連結
URDF Importer ROS 生態標準格式匯入 說明｜教學
USD → URDF Exporter 反向匯出給 ROS 工具鏈 說明｜教學
MJCF Importer MuJoCo 格式匯入 說明｜教學
Onshape Importer 雲端 CAD 直接匯入 說明
CAD Converter STEP 等 CAD 格式轉 USD 說明
ShapeNet Importer ShapeNet 資料集物件 說明｜教學

各格式的能力比較見 Formats。

2. 機器人設定（Robot Setup）#

檢查與編輯工具：

  • Robot Inspector——檢視機器人結構與屬性。
  • Robot Poser——擺姿勢、存姿態。
  • Robot Assembler——把手臂＋夾爪、車＋手臂組裝成一台。
  • Gain Tuner——關節增益（stiffness/damping）調校。
  • 自碰撞偵測、網格合併工具。
  • Joint Inspector／Physics Inspector——關節極限、驅動參數即時檢視。
  • Asset Validation（資產驗證）、Asset Transformer（批次規則化轉換資產，含 API 與規則參考）。
  • 資產的標準結構規範：Asset Structure。

Robot Setup 教學系列（13 篇）——從零 rig 一台機器人的主線教材：

  1. Stage 設定 → 2. 組裝簡單機器人 → 3. 基礎 Articulation → 4. 加相機與感測器 → 5. Rig 移動機器人 → 6. 設定機械手臂 → 7. 配置機械手臂 → 8. 產生機器人設定檔 → 9. 夾取放置範例 → 10. 閉鏈結構 → 11. 關節增益調校 → 12. 資產最佳化 → 13. 腿式機器人 rigging（locomotion policy）。

另有 OpenUSD 與調校最佳實務系列（7 篇）：以靈巧手為例走過資產結構、碰撞對、關節驅動與增益調校。

3. 機器人模擬與控制（Robot Simulation）#

  • Articulation Controller——關節層控制核心（Python 用法見《完整文檔》7.3 節與《Script Editor 教學》第 6 課）。
  • 移動機器人控制器——差速、Ackermann 等。
  • Surface Gripper（吸附式夾爪）、Grasp Editor。
  • RL 策略範例——在 Isaac Sim 內執行訓練好的策略（四足、人形等）。
  • 模擬技巧與核心概念連結集。
  • Newton Actuators 教學——Newton 引擎的致動器：Python 設定、USD 撰寫、OmniGraph 驅動。

已棄用但仍在文檔中的 Motion Generation（Lula）：總覽——RMPflow、RRT、運動學求解器、軌跡生成等，新專案請改用下方實驗性方案。

4. 運動規劃（實驗性 Robot Motion）#

  • Motion Generation (Experimental)：場景互動、控制器與 RobotState、軌跡規劃與執行。
  • cuMotion 整合：GPU 加速運動規劃——機器人設定、世界介面、RMPflow、圖式規劃器、軌跡生成／最佳化教學。
  • PINK 整合：微分 IK——IK 控制器、多任務。

疑難排解#

Robot Setup Troubleshooting——匯入後爆炸、關節抖動、增益不穩等經典問題。
