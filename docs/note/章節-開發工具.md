## OmniGraph（視覺化編程）

節點圖框架，GUI 端機器人控制器與 ROS 2 Bridge 的底層。學習路徑：

1. [OmniGraph 核心概念](https://docs.omniverse.nvidia.com/extensions/latest/ext_omnigraph/getting-started/core_concepts.html)與[介面](https://docs.omniverse.nvidia.com/extensions/latest/ext_omnigraph/interface.html)（Omniverse 文檔）。
2. [Isaac Sim OmniGraph 教學](https://docs.isaacsim.omniverse.nvidia.com/latest/omnigraph/omnigraph_tutorial.html)——建立差速機器人控制圖。
3. [以 Python 腳本操作 OmniGraph](https://docs.isaacsim.omniverse.nvidia.com/latest/omnigraph/omnigraph_scripting.html)——程式化建圖與設值。
4. 自訂節點：[Python 節點](https://docs.isaacsim.omniverse.nvidia.com/latest/omnigraph/omnigraph_custom_python_nodes.html)｜[C++ 節點](https://docs.isaacsim.omniverse.nvidia.com/latest/omnigraph/omnigraph_custom_cpp_nodes.html)｜[IPC 節點](https://docs.isaacsim.omniverse.nvidia.com/latest/omnigraph/omnigraph_custom_ipc_nodes.html)。
5. [常用捷徑](https://docs.isaacsim.omniverse.nvidia.com/latest/omnigraph/omnigraph_shortcuts.html)——Tools > Robotics > OmniGraph Controllers 系列快速產生器。

## 開發流程建議

1. **探索期**：Script Editor＋Commands Tool（看 GUI 操作對應什麼 API）。
2. **迭代期**：Extension＋熱重載，或 VS Code＋Python Server。
3. **量產期**：Standalone 腳本（無頭、可 CI），需要時用 Tracy 剖析效能。
