安裝與快速入門#

整理自官方 System Requirements、Installation 與 Dependency Graph 三章。

系統需求#

項目需求
作業系統Ubuntu 24.04；Windows（原生／WSL）
Python≥ 3.10（pip 安裝時）
驅動pip：與本機 PyTorch 相容的 NVIDIA 驅動；容器：26.05 容器基於 PyTorch 26.04 容器，見 PyTorch 容器 Release Notes
建議硬體64 位元 x86（ARM64 亦支援）；GPU：Blackwell（B100／RTX PRO 6000）、Hopper（H100／H200）、Ada（RTX 40xx）、Ampere（A100／A30／A40／RTX 30xx）、Turing（T4）

官方建議：要拿到全部效能改進，優先使用 NGC 的 PhysicsNeMo 容器——所有先決條件與相依都已就緒。

兩條安裝路徑怎麼選#

情境建議
想訓練模型或跑官方範例容器（現成環境）或 pip／uv（先裝 PyTorch 再裝 PhysicsNeMo）皆可
領域專家，想使用／改造既有模型容器或 pip／uv，並從 GitHub 下載 examples 作為起點
想貢獻 PhysicsNeMo 本體從原始碼安裝，見官方 Customizing PhysicsNeMo

方式一：NGC 容器#

PhysicsNeMo NGC 容器基於 NVIDIA PyTorch 容器，預裝全部相依。需先安裝 Docker Engine 與 NVIDIA Container Toolkit：

# 拉取映像（tag 見 NGC 頁面，如 26.05）
docker pull nvcr.io/nvidia/physicsnemo/physicsnemo:<tag>

# 啟動 shell 並掛載目前目錄
docker run --shm-size=1g --ulimit memlock=-1 --ulimit stack=67108864 \
           --runtime nvidia -v ${PWD}:/workspace \
           -it --rm nvcr.io/nvidia/physicsnemo/physicsnemo:<tag> bash

多 GPU 執行加上 --gpus all。HPC 叢集可改用 Singularity／Apptainer：

singularity build PhysicsNeMo.sif docker://nvcr.io/nvidia/physicsnemo/physicsnemo:<tag>
singularity run --writable --nv PhysicsNeMo.sif

方式二：pip／uv 安裝#

先依 PyTorch 官方指南安裝 PyTorch，再安裝 PhysicsNeMo：

pip install nvidia-physicsnemo
python -c "import physicsnemo; print('PhysicsNeMo version:', physicsnemo.__version__)"

要一併解析 CUDA 對應的 PyTorch 與 RAPIDS 套件，用 cu12／cu13 extra，並可疊加功能 extras：

# CUDA 13 + 神經網路加值套件
pip install "nvidia-physicsnemo[cu13,nn-extras]"

# 物理資訊工作流程（安裝 physicsnemo.sym 符號數學）
pip install "nvidia-physicsnemo[sym]"

開發或跑 repo 範例建議用 uv：

git clone https://github.com/NVIDIA/physicsnemo.git
cd physicsnemo
uv sync --extra cu13 --extra nn-extras
uv run python -c "import physicsnemo; print(physicsnemo.__version__)"

可選 extras 相依群組#

PhysicsNeMo 把相依拆成「有界的核心」＋「明確選用的 extras」。核心安裝包含 PyTorch、NumPy、ONNX、pandas、Warp、Hydra／OmegaConf 等通用套件；特殊能力透過具名群組選裝：

Extra內容典型用途
cu12／cu13CUDA 對應的 PyTorch／TorchVision + RAPIDS、CuPy、DALI選定 CUDA 12 或 13 後端（兩者互斥）
natten-cu12/13預建的 NATTEN wheel鄰域注意力（neighborhood attention），免自行編譯
transformer-engine-cu12/13CUDA 對應的 Transformer Enginefp8 訓練／Tensor Core 最佳化（需搭配同版 CUDA extra）
utils-extrasW&B、MLflow、line profiler、VTK、STL實驗追蹤、profiling、支援資料格式
mesh-extrasmatplotlib、PyVista、PyACVD、VTK網格處理、檢視與視覺化（配 PhysicsNeMo-Mesh）
nn-extrasSciPy 與共用工具physicsnemo.nn 層的可選功能
model-extrasnn-extras + PyVista + VTK模型套件的可選功能
datapipes-extrasTFRecord、Dask、NetCDF4、Xarray、Zarr大型科學資料集與資料管線格式
uq-extrasGPyTorch不確定性量化（高斯過程）
gnnsPyTorch Geometric 相關 + model extras圖神經網路工作流程
symSymPyphysicsnemo.sym 符號 PDE／物理殘差

完整清單見 pyproject.toml。GNN 工作流程搭 CUDA 後端的組合範例：pip install "nvidia-physicsnemo[cu13,gnns]"。

Hello World：驗證安裝#

import torch
from physicsnemo.models.mlp.fully_connected import FullyConnected

model = FullyConnected(in_features=32, out_features=64)
input = torch.randn(128, 32)
output = model(input)
print(output.shape)   # torch.Size([128, 64])

成功輸出即代表安裝完成。許多官方範例需要額外套件——查看各範例目錄的 README.md 或 requirements.txt。
