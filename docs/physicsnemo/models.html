模型架構#

整理自官方 Model Architectures 章節與 Models API Reference。

PhysicsNeMo 提供五大類經過微調與最佳化的模型架構積木（transformer 層、GNN 層等），既可直接配置使用，也可作為自研模型的建材。每個架構都配有端到端的訓練配方（training recipe）作為參考實作。選型的第一原則是資料形態：

資料形態適合家族代表架構
非結構化網格／粒子／圖GNNMeshGraphNet、GraphCast、X-MeshGraphNet
點雲、複雜幾何、長程交互TransformerTransolver、GeoTransolver、DPOT
結構化網格上的 PDE 函數族神經算子FNO、AFNO、SFNO、DeepONet、DoMINO
不適定反問題／生成任務擴散模型Diffusion UNet（EDM）、Diffusion Transformer（DiT）
規則 3D 體素網格Voxel／CNN3D UNet、FigConvUNet

圖神經網路（GNN）#

GNN 的優勢是直接在非結構化／不規則網格上操作——這是相對 CNN 等網格式模型的關鍵差異。PhysicsNeMo 對 MeshGraphNet 與 GraphCast 提供頑健支援，應用橫跨流體、結構力學到全球天氣預報：

訓練配方架構重點
渦流脫落（2D 非定常）MeshGraphNetTFRecord 時序資料；DDP 多 GPU
分散式淺水方程GraphCast二十面體網格即時生成；分散式訊息傳遞
XAeroNet（X-MeshGraphNet）X-MeshGraphNet圖分割 + halo 區域的多尺度擴展，可到 1 億+ cell
拉格朗日粒子模擬MeshGraphNet連結性隨時間變化的點雲式資料
中期天氣預報GraphCast自迴歸；gradient checkpointing、fused SiLU、多步 rollout

官方另有 MeshGraphNet 實務教學。注意：GNN 後端已從 DGL 遷移至 PyTorch Geometric，遷移指南見 Switching from DGL to PyG；安裝用 gnns extra。

Transformer#

Transformer 擅長物理問題中關鍵的長程交互作用，把輸入視為無序點集合，比網格式模型更能適應複雜幾何。代表是 Transolver（PhysicsAttention 作用於表面網格 + 即時計算的 SDF 值），用於外氣動力等 CFD 代理模型：

訓練配方架構重點
Transolver 外氣動力TransolverDDP＋領域平行；Transformer Engine 加速 25%，支援 fp8 訓練／推論
Transolver Darcy 流Transolver2D 影像資料；DDP
DPOT 算子 TransformerDPOT譜域 Fourier attention 打造 PDE 基礎模型

此外 GraphCast、AFNO 與多數擴散模型也大量使用注意力層。Diffusion Transformer（DiT）骨幹特別模組化——注意力、tokenization／de-tokenization 後端都可換成你自訂的 Module。

神經算子（Neural Operators）#

神經算子學習無限維函數空間之間的映射——等於一次學會一整族 PDE 的解，且與網格解析度無關。主要架構：FNO、DeepONet、物理資訊變體 PINO，以及新一代泛化能力出色的 DoMINO：

訓練配方架構重點
磁流體動力學 PINOFNO（物理資訊）譜方法＋有限差分梯度；DDP
DoMINO 外氣動力DoMINOCAD 點雲＋局部 stencil，免模擬網格；自訂 Warp kernel 投影快 10 倍+；DDP＋領域平行
中期天氣預報（Makani）SFNO球面 FNO；空間模型平行（模型＋資料同時切分）
Darcy 流 DeepONetDeepONet（FNO branch＋FC trunk）自動微分物理資訊；支援非結構化隨機取樣

擴散模型（Diffusion Models）#

PhysicsNeMo 提供現成的擴散骨幹（EDM／ADM 風格 U-Net 變體、DiT）＋完整配套工具——取樣器、損失函數、preconditioner、噪聲排程器——組成 physicsnemo.diffusion 專用套件。擴散方法特別適合不適定反問題：超解析、降尺度、幾何直接生成：

訓練配方架構重點
StormCast 對流尺度預報Diffusion UNetEDM 取樣器；fused ops、Apex GroupNorm、bf16 AMP
CorrDiff 公里級降尺度Diffusion UNetEDM；multi-diffusion 可擴展至 2000×2000+ 像素大域；patch-wise 梯度累積
全波形反演（FWI）Diffusion UNet＋GFN物理資訊 Diffusion Posterior Sampling（DPS）
拓撲最佳化 TopoDiffDiffusion UNet＋自訂編碼器自訂 DDPM＋DPS，生成滿足工程約束的新拓撲

API 詳見 PhysicsNeMo Diffusion（Introduction／Model Backbones／Noise Schedulers／Preconditioners／Metrics／Samplers／Multi-Diffusion／Utilities 八節）。

Voxel 模型（3D U-Net 家族）#

把 3D 物理域當作規則體素網格，將網格式物理問題轉化為 3D 影像任務，直接套用高度最佳化的電腦視覺架構（U-Net 編碼—解碼＋跳接）：

訓練配方架構重點
資料中心熱流數位孿生3D UNet有限差分物理資訊；結構化 3D 體積場；DDP
FigConvUNet 外氣動力FigConvUNetCAD／網格點雲輸入；AMP（fp16/bf16）
3D Gray-Scott 瞬態系統RNN＋UNet4D 資料（時間×3D 體積）
