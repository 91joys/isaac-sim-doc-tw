建議學習路徑

  1. 總覽文檔：先理解 PhysicsNeMo 的定位——PyTorch 之上的 SciML 工具箱，而非黑盒求解器。
  2. 安裝與快速入門：pip install nvidia-physicsnemo 或 NGC 容器，跑起第一個 FullyConnected 模型。
  3. 訓練工作流程：掌握「datapipe → model → loss → optimizer」的標準訓練配方與 checkpoint／日誌工具。
  4. 模型架構：認識五大模型家族，依你的資料形態（網格／點雲／結構化網格）選擇架構。
  5. 物理引導與 PINNs：用 physics loss 與 PhysicsInformer 把守恆律與 PDE 殘差加進訓練。
  6. 分散式與領域平行：從單 GPU 到多節點——DDP 起步，資料放不下時上 ShardTensor。
  7. 依方向深入：PhysicsNeMo-Mesh（幾何前處理）、效能最佳化（profiling／compile）、領域套件（CFD／氣象推論與資料策展）。
  8. 動手做：到範例目錄挑一個貼近你領域的官方配方，複製後改造成自己的專案。
