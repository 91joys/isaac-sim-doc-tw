物理引導（Physics-guided）與 PINNs#

整理自官方 Physics-guided 與 PINNs 教學 章節，及 physicsnemo.sym API。

給模型加入歸納偏置（inductive bias）能顯著改善泛化能力。Physics-AI 有兩條路：一是選擇天生適合物理資料的架構（FNO 用 FFT、MeshGraphNet 直接吃模擬網格、DoMINO 的類數值方法 stencil）；二是把控制方程當作損失函數，正則化模型預測使其遵守物理。本頁講第二條路。

v2.0 重要變更：舊獨立套件 PhysicsNeMo-Sym 已上游整併為 physicsnemo.sym 命名空間（原 repo 已封存），以 pip install "nvidia-physicsnemo[sym]" 安裝。舊 Solver／Domain／Constraint／Geometry 抽象已改為顯式 PyTorch 迴圈寫法，對照見 v2.0 Migration Guide。

最簡單的物理損失：代數約束#

不是所有物理約束都需要 PDE。以分子動力學為例：平衡系統中所有分子受力總和應為零，一行就能加進損失：

out = model(node_features, edge_features, input_graph)   # (N,3) 每個分子的力
loss_data = torch.nn.functional.l1_loss(out, true_out)             # 資料損失
loss_physics = torch.mean(out).abs()                               # 合力為零的約束
loss = loss_data + 0.001 * loss_physics                            # 權重（拉格朗日乘子）
loss.backward()

完整範例見分子動力學範例。

PDE 損失：PhysicsInformer#

控制方程含 PDE 時，計算殘差需要空間／時間梯度，而梯度算法取決於資料形態。physicsnemo.sym.eq.phy_informer.PhysicsInformer 統一了這件事——在點雲、網格（grid）、圖／網格（mesh）上計算 PDE 殘差，梯度方法可選：

grad_method方法適用
"autodiff"自動微分點雲／PINN；精確但需要輸入座標 requires_grad
"finite_difference"有限差分結構化網格；快速
"meshless_finite_difference"無網格有限差分散點資料
"least_squares"最小平方非結構化網格／mesh
"spectral"譜微分週期域、FNO 類模型

方程本身以 physicsnemo.sym.eq.pde.PDE 子類用 SymPy 符號式定義——內建 Navier–Stokes、擴散、波動等常見方程，也支援自訂 PDE。方程只寫一次，正問題與反問題都能重用。

PINN 工作流程#

PhysicsNeMo 把 PINN 訓練寫成普通的 PyTorch 訓練迴圈，由五個顯式積木組成：

  1. 網路：任何 torch.nn.Module，典型是 physicsnemo.models.mlp.fully_connected.FullyConnected（座標 → 場變數）。
  2. PDE 定義：physicsnemo.sym.eq.pde.PDE 子類，用 SymPy 宣告控制方程。
  3. PhysicsInformer：給定 PDE 與 grad_method，在網路輸出上計算導數與方程殘差。
  4. 損失與優化器：標準 PyTorch 損失（資料項＋物理殘差項）＋任何 torch.optim 優化器。
  5. 配置：Hydra（@hydra.main）讀一個小小的 config.yaml，沒有框架特殊包裝。

與其他 PhysicsNeMo 工作流程完全同構：同一套 physicsnemo.distributed 多 GPU 初始化、同一套日誌／checkpoint 工具——物理資訊只是損失裡多一項殘差。

資料 × 物理的混合訓練#

物理損失也可以加在資料驅動模型上（物理引導、physics-informed 的 FNO／DeepONet 等）：

  • Darcy 物理資訊範例：同一問題分別用 PINO（譜梯度）與 DeepONet（autodiff）兩種方式物理化。
  • 磁流體 PINO：譜方法＋有限差分混合梯度。
  • Stokes 流 MeshGraphNet：訓練後用物理資訊推論修正提升精度。

理論背景：物理損失、可微物理與兩者的取捨，可對照本站 PBDL：物理損失 與 PBDL：可微物理 章節。
