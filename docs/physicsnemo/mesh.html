PhysicsNeMo-Mesh：GPU 加速網格模組#

整理自官方 PhysicsNeMo-Mesh User Guide（四章）與 Mesh API Reference（18 節）。v2.0（26.03）新增模組。

PhysicsNeMo-Mesh 是 GPU 加速、基於 PyTorch 的網格模組，為 Physics-AI 工作流程而生。「mesh」對不同社群意義不同——CFD／FEM 想到體積網格、圖學想到表面網格、電腦視覺想到點雲、機器人學想到曲線——PhysicsNeMo-Mesh 用單一 API 全部處理：點雲、曲線、曲面、體積，並整合 PyTorch autograd。

pip install "nvidia-physicsnemo[cu13,mesh-extras]"

能力面涵蓋：離散微積分、微分幾何、細分（subdivision）、修復（repair）、空間查詢、拓撲與 I/O。動手教學在 examples/minimal/mesh。

章節內容
Concepts and Data Model維度通用性、五欄位 Mesh tensorclass、為何需要任意秩（arbitrary-rank）張量場、單純形（simplicial）限制

在框架中的定位#

  • 資料前處理搬上 GPU：傳統上網格前處理（取樣、SDF、重網格化）在 CPU 用 VTK／trimesh 做，常成為訓練瓶頸；PhysicsNeMo-Mesh 讓這些操作以 PyTorch 張量在 GPU 上直接進行，可嵌入 datapipe 甚至反向傳播。
  • 多資料集訓練基礎：26.05 的統一外氣動配方用它支援多資料集網格訓練。
  • 與 Warp 互補：Warp 提供 kernel 級幾何原語（BVH、HashGrid），PhysicsNeMo-Mesh 提供張量級網格資料模型與演算法。
