PhysicsNeMo Distributed：分散式與領域平行#

整理自官方 PhysicsNeMo Distributed 與 Domain Parallelism（三篇教學）章節。

科學 AI 的規模化與 LLM 平行有本質差異——不只因為模型不同，資料也不同：SciML 處理的不是有序 token 或像素影像，而是網格、點雲、巨型連通圖、高解析度 3D 資料。而多數 SciML 模型（GNN、神經算子）參數在 10 億以下，讓 DeepSpeed／FSDP 式的模型平行對小模型無關緊要；真正主流的是兩種平行：

策略切什麼什麼時候用
資料平行（Data Parallelism）batch 切到多 GPU，每卡至少一個完整樣本模型與單一樣本放得進一張卡，想放大 batch 與吞吐量——用 PyTorch DDP
領域平行（Domain Parallelism）單一訓練樣本切到多 GPU 協同處理單一樣本（高解析 3D 場、億級 cell 網格）一張卡放不下

Pipeline 平行對推論工作負載也很有價值；Megatron 式 tensor 平行則保留給真正的超大參數模型。

DistributedManager：一切的起點#

physicsnemo.distributed.DistributedManager 是單例（singleton），統一偵測與配置分散式環境（torchrun／mpirun／SLURM 都認得）：

from physicsnemo.distributed import DistributedManager

DistributedManager.initialize()
dist = DistributedManager()

dist.rank          # 全域 rank
dist.local_rank    # 本節點 rank（給 device_ids）
dist.world_size    # 總程序數
dist.device        # 此程序該用的 torch.device
dist.distributed   # 是否為多程序執行

搭配 DDP 的完整範例見訓練工作流程 § 分散式。另有可微分集合通訊（physicsnemo.distributed.autograd）：gather_v、all_gather_v、scatter_v、indexed_all_to_all_v——專為不規則資料（每 rank 大小不一的圖分割）設計，梯度可以反向流過通訊操作。

資料怎麼分散#

SciML 資料集常以 TB 計，資料載入是首要瓶頸。官方建議的高效能儲存格式：Zarr、Xarray（雲原生、可平行讀寫），以及 HDF5（h5py）與 NetCDF。點雲／網格／圖資料可直接用 physicsnemo.datapipes 的專用管線。

ShardTensor：通用領域平行#

部分模型原生支援領域平行（SFNO、MeshGraphNet）；其餘模型靠 ShardTensor——PhysicsNeMo 開發中的通用領域平行機制，把張量沿空間維度切分到 DeviceMesh 上，自動處理 halo 交換，且可與 FSDP／DDP 組合。官方三篇教學：

教學內容
Domain Parallelism and Shard Tensor領域平行是什麼、何時需要、PhysicsNeMo 如何支援——概念總覽
Implementing New Layers for ShardTensor尚未支援的操作如何自己擴展——用 PyTorch 與 PhysicsNeMo 工具實作新的分片層
Domain Decomposition, ShardTensor, and FSDP Tutorial端到端合成資料範例：領域平行 × 資料平行（FSDP）組合使用

API 見 physicsnemo.domain_parallel（ShardTensor 本體）與 physicsnemo.distributed。

分散式詞彙速查#

術語意義
CollectivesGPU 間張量通訊的原語（all-reduce、all-gather…）；PhysicsNeMo 提供可微分版本
DDP（DistributedDataParallel）PyTorch 資料平行：每 rank 複製完整模型、餵不同 batch、all-reduce 同步梯度
FSDPFully Sharded Data Parallel：把參數／梯度／優化器狀態切分到各 rank，給大模型省記憶體
DeviceMeshPyTorch 的多維裝置網格抽象，ShardTensor 建構於其上；可同時表達「資料平行 × 領域平行」兩個維度
Halo 區域領域切分邊界上需要與鄰居交換的重疊區——卷積、訊息傳遞都需要
DeepSpeed／MegatronLLM 圈的 ZeRO 分片與 tensor 平行方案，SciML 小模型通常用不到
