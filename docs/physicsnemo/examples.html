怎麼挑範例#

  1. 照資料形態：結構化網格 → Darcy FNO；非結構化網格 → vortex shedding MGN；點雲 → DoMINO；球面／全球 → GraphCast。
  2. 照方法：純物理（無資料）→ LDC PINN；資料＋物理 → darcy_physics_informed；生成式 → CorrDiff／TopoDiff。
  3. 照規模：單卡入門 → introductory 七例；多 GPU → GraphCast／Transolver；超大樣本 → XAeroNet、swe_distributed_gnn。
  4. 每個範例目錄含 README、requirements.txt 與 Hydra 配置——複製整個目錄開始改造即可。
