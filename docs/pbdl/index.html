建議閱讀路徑

  1. 總覽（緒論）：先建立「監督式 → 物理損失 → 可微分物理」由鬆到緊的整合光譜，以及正／反問題的分類。
  2. 神經代理與算子：從最傳統的監督式學習入手，理解資料驅動代理的能與不能。
  3. 物理損失項（PINN） → 可微分物理：對照兩種「把物理放進訓練」的做法，理解為何 DP 能泛化到整類反問題。
  4. 依興趣深入：機率式與生成學習（不確定性）、強化學習（控制）、改良梯度（優化）。
