1. 什麼是 PBDL#

原文連結

物理化深度學習（Physics-Based Deep Learning, PBDL）研究的是：如何把物理模型（尤其是以偏微分方程 PDE 描述的數值模擬）與深度學習結合起來，一起解決科學與工程中的模擬問題。核心理念不是「用神經網路取代數值方法」，而是橋接兩個世界——讓深度學習借助 PDE 所編碼的領域知識，也讓傳統求解器借助神經網路的靈活與速度。

原書涵蓋的 PBDL 方法類別包括：

  • 神經代理與算子（Neural surrogates & operators）：以資料訓練快速的替身模型。
  • 物理損失項（Physical loss terms，PINN）：把 PDE 殘差當成約束放進損失。
  • 可微分物理（Differentiable physics）：讓求解器可反向傳播，與網路緊密耦合。
  • 機率式／生成學習：擴散模型、流匹配等，處理不確定性與多解。
  • 強化學習：用於控制類問題。
  • 改良梯度：針對物理問題病態梯度的優化技巧。
  • 進階架構：適用非結構化網格的神經網路。

作者群把這些方法定位為新興「科學基礎模型（scientific foundation models）」的基石。

2. 為何橋接模擬與學習#

純資料驅動的深度學習雖靈活，但常忽略我們早已透過數百年物理研究掌握的規律；傳統數值模擬雖精確，卻昂貴且難以處理反問題與參數化不確定性。PBDL 主張把兩者的長處合起來：

  • 用物理模型提供歸納偏置（inductive bias），減少對資料量的依賴、提升泛化與可信度。
  • 用神經網路加速昂貴的求解、修正數值誤差，或補足模型中未解析的項（歷史上稱為 closure、homogenization、parametrization 問題）。

神經網路也能扮演求解器修正器（solver corrector）：在每一步模擬後修正離散化誤差或未解析的物理，這正是可微分物理最有威力的用法之一。

3. 三大整合方式#

原書依「物理模型與神經網路耦合的緊密程度」把方法分成三大類，由鬆到緊：

方式物理如何進入耦合程度對應章節
監督式（Supervised）只用（真實或模擬產生的）資料訓練網路，訓練時不再與物理互動——經典機器學習做法鬆神經代理與算子
物理損失（Loss-terms / PINN）把物理動力學寫成損失中的可微分項，用梯度反覆評估 PDE 約束中物理損失項
可微分物理（Differentiable / Hybrid）完整的物理模擬與網路輸出交錯運算，需要可微分求解器；耦合最緊緊可微分物理

這條「由鬆到緊」的光譜是全書的主軸：越往右，網路越能與真實的物理過程互動，泛化能力與物理一致性通常越好，但實作也越需要對數值方法的理解。

4. 正問題與反問題#

除了整合方式，問題本身也可分為兩類：

  • 正問題（Forward problems）：已知系統與初始／邊界條件，預測狀態或其時間演化（例如給定初始流場，預測下一時刻）。
  • 反問題（Inverse problems）：由觀測回推參數化或狀態（例如由稀疏量測反推材料參數、初始條件或最佳控制力）。反問題通常更難、更病態，也是 PBDL（尤其可微分物理）最能發揮的地方。

5. 常用 PDE 與符號慣例#

全書以兩個 PDE 作為反覆出現的範例：

Burgers 方程#

一維、含黏滯項的非線性對流方程，是研究衝擊波與非線性的最小範例：

∂u/∂t + u · ∂u/∂x = ν · ∂²u/∂x²

其中 u 為速度場、ν 為黏滯係數。它足夠簡單可快速實驗，又保有非線性帶來的困難。

Navier-Stokes 方程#

描述不可壓縮流體的動量與連續方程，是流體模擬的核心：

∂u/∂t + (u · ∇)u = −(1/ρ)∇p + ν∇²u , ∇ · u = 0

其中壓力投影（求解 Poisson 方程 ∇²p = …）是可微分物理中隱式微分的經典場景。

符號慣例#

符號意義
u待求解的物理場（如速度）；下標表偏導，如 u_t、u_xx
P / P_i物理求解器（或其分解出的算子），u(t+Δt)=P(u(t))

7. 範圍與實作環境#

原書聚焦於以稠密場（dense field-based，Eulerian）表示的模擬，範例以下列框架實作：

  • PyTorch、TensorFlow、JAX：主流深度學習與自動微分框架。
  • ΦFlow（PhiFlow）：作者團隊開發、面向可微分模擬的框架。

原書刻意不涵蓋 Lagrangian（粒子式）方法與真實世界實驗資料，以聚焦核心概念。每章多附可直接執行的 Jupyter Notebook。
