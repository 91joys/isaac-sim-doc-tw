教學與範例#

對應官方 Tutorials 章節。官方提供兩套可在 Colab 直接開啟的教學筆記本，加上 newton/examples/ 範例庫。筆記本偵測到 Colab 環境時，第一個 code cell 會自動從 PyPI 安裝依賴。

教學 00：Introduction to Newton Physics#

官方頁面｜ 在 Colab 開啟

第一手實作走查，內容依序為：

  1. Setup and Imports：安裝與匯入 warp、newton。
  2. Building a Model with ModelBuilder：加剛體、形狀與地面，finalize()。
  3. Creating States and Control：配置 state_0／state_1、control、contacts。
  4. Setting Up the Solver：建立求解器並理解 step() 的輸入輸出。
  5. Configuring the Simulation Loop：substep 結構與雙 state 交換。
  6. GPU Acceleration with CUDA Graphs：用 wp.ScopedCapture() 把整個模擬迴圈捕捉成 CUDA graph，大幅降低 kernel 啟動開銷： 
    if wp.get_device().is_cuda:
    with wp.ScopedCapture() as capture:
        simulate()
    graph = capture.graph      # 之後以 wp.capture_launch(graph) 重播
  7. Visualization／Running the Simulation：接上 Viewer 觀看結果。

教學 01：Newton Robotics 101——從概念到操作#

官方頁面｜ 在 Colab 開啟

機器人向的完整路線圖：

  1. From ModelBuilder to Model：核心概念與 Warp 基礎複習。
  2. Solver and Simulation Loop：求解器與模擬迴圈。
  3. Performance: CUDA Graph Capture：效能加速。
  4. Load a Robot and Apply Joint Forces：載入機器人、施加關節力。
  5. Joint Targets with Control.joint_target_q：PD 目標控制。

  7. Manipulation：Franka Pick-and-Place：IK＋夾爪控制完成取放任務。

本站實戰教學：可微分物理#

本站另備有兩篇中文實戰教學，皆以官方 diffsim_ball 範例為主軸：

  • 《可微分物理實戰教學：讓梯度學會丟球》——七課帶你改編官方範例，用 wp.Tape() 把梯度反向傳播穿過 288 步模擬，以梯度下降優化球的初速度命中目標，含完整可執行程式與常見陷阱。
  • 《example_diffsim_ball.py 逐行精讀》——對官方原始碼逐段解說，每段都附「數字帶入」實算（時步換算、半隱式歐拉手算一步、接觸力估算、中央差分驗證梯度…），適合想把每一行都吃透的讀者。

範例庫（newton/examples）#

安裝 newton[examples] 後：

python -m newton.examples            # 啟動預設 basic_pendulum，側欄可切換其他範例
python -m newton.examples --list     # 列出全部範例
python -m newton.examples robot_anymal_c_walk   # RL 策略推論（需 torch extra）

範例涵蓋：基礎剛體（pendulum）、機器人（ANYmal、humanoid、Franka）、布料與軟體、粒子／MPM、感測器（example_sensor_contact、example_sensor_imu、example_sensor_tiled_camera）與 diffsim 可微分模擬系列。原始碼在 GitHub。
