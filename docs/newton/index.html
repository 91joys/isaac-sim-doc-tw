建議學習路徑

  1. 總覽文檔：先理解 Newton 的定位（引擎 vs. 模擬器）與七大核心物件。
  2. 安裝與快速入門：pip install "newton[examples]"，跑起第一個內建範例與快速入門程式。
  3. 核心概念：掌握 ModelBuilder → Model → State 的資料流，與多世界（Worlds）平行模擬。
  4. 求解器：依需求選擇求解器——機器人 RL 用 MuJoCo、布料軟體用 VBD／Style3D、可微分用 SemiImplicit／Featherstone。
  5. 教學與範例：跟著官方 Colab 筆記本完成 Franka 手臂 IK 取放。
  6. 可微分物理教學：親手做一個梯度穿過整段模擬的最佳化 demo，體驗 Newton 最獨特的能力。
  7. 依方向深入：碰撞、感測器、視覺化，或到 FAQ 了解 Isaac Lab 整合。
