安裝與快速入門#

對應官方 Installation 章節。本頁涵蓋從 PyPI 安裝的建議方式；從原始碼安裝或使用 uv 請見官方 Development 指南。

系統需求#

需求最低備註
Python3.10建議 3.11+（3.10 安裝 imgui_bundle 可能出錯）
作業系統Linux（x86-64、aarch64）、Windows（x86-64）、macOS（僅 CPU）macOS 無 GPU 加速
NVIDIA GPUCompute Capability 5.0+（Maxwell）GeForce GTX 9xx 以上皆可
NVIDIA 驅動545 以上（CUDA 12）建議 550 以上（CUDA 12.4）以獲得最佳效能
CUDA12、13不需安裝本機 CUDA Toolkit——Warp 自帶 runtime

ARM64 Linux（Jetson Thor、DGX Spark 等）：安裝 examples 附加依賴時需要 X11 開發函式庫來編譯 imgui_bundle：
sudo apt-get install -y libx11-dev libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev libgl1-mesa-dev

安裝 Newton#

基本安裝（唯一必要依賴是 NVIDIA Warp）：

pip install newton

加裝 examples 附加集以執行內建範例（含模擬與視覺化依賴）。建議在虛擬環境中安裝：

# Windows（PowerShell）
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install "newton[examples]"

# macOS / Linux
python -m venv .venv
source .venv/bin/activate
pip install "newton[examples]"

附加依賴集（extras）#

集合用途
sim模擬依賴，含 MuJoCo（使用 SolverMuJoCo 必裝）
importers資產匯入與網格處理
remesh重新網格化（Open3D、pyfqmr）
examples執行範例（含視覺化；包含 sim＋importers）
torch-cu12／torch-cu13PyTorch（CUDA 12.8+／13），執行 RL 策略範例（包含 examples）
notebookJupyter＋Rerun 視覺化（包含 examples）
dev／docs開發測試／建置文檔

集合有遞移包含關係（examples ⊃ sim＋importers），只需安裝最符合用途的那一組。

執行內建範例#

# 啟動預設 basic_pendulum 範例（可在側欄瀏覽其他範例）
python -m newton.examples

# 列出全部範例
python -m newton.examples --list

執行 RL 策略推論範例——先用 nvidia-smi 查驅動支援的 CUDA 版本（輸出右上角），選擇對應的 extra：

pip install newton[torch-cu12] --extra-index-url https://download.pytorch.org/whl/cu128
python -m newton.examples robot_anymal_c_walk

torch-cu12 安裝的是 CUDA 12.8 版 PyTorch。若驅動只支援 CUDA 12.4／12.5，請改手動安裝：
pip install "newton[examples]" 後接 pip install torch==2.6.0 --extra-index-url https://download.pytorch.org/whl/cu124

快速入門：第一個模擬#

只用基本安裝（pip install newton）即可執行——一顆球落到地面：

import warp as wp
import newton

# 建構模型
builder = newton.ModelBuilder()
body = builder.add_body(
    xform=wp.transform((0.0, 1.0, 0.0), wp.quat_identity()),
    mass=1.0,
)
builder.add_shape_sphere(body, radius=0.25)
builder.add_ground_plane()
model = builder.finalize()

# 建立求解器並配置狀態
solver = newton.solvers.SolverXPBD(model)
state_0 = model.state()
state_1 = model.state()
control = model.control()
contacts = model.contacts()

newton.eval_fk(model, model.joint_q, model.joint_qd, state_0)

# 步進模擬
for step in range(120):
    state_0.clear_forces()
    model.collide(state_0, contacts)
    solver.step(state_0, state_1, control, contacts, 1.0 / 60.0)
    state_0, state_1 = state_1, state_0

進一步：用 SolverMuJoCo（需 pip install "newton[sim]"）把一個機器人模板複製成 1024 個平行世界，GPU 一次步進全部：

template = newton.ModelBuilder()
template.add_mjcf("humanoid.xml")

builder = newton.ModelBuilder()
builder.replicate(template, world_count=1024)
builder.add_ground_plane()
model = builder.finalize()

solver = newton.solvers.SolverMuJoCo(model)

相容性與支援#

  • Newton 遵循語意化版本；穩定版文檔對應 1.3.0。
  • CUDA 12 與 13 皆支援；Warp 自帶 CUDA runtime，無需本機 Toolkit。
  • 平台測試組態、版本策略與棄用政策見官方 Compatibility and Support。
