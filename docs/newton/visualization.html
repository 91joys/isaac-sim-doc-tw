視覺化（Visualization）#

對應官方 Visualization 章節。Newton 內建的視覺化以除錯為目的；需要高品質即時渲染時，官方建議搭配 Isaac Lab，或輸出時間取樣 USD 到 Omniverse／Isaac Sim 檢視。

七種 Viewer 選擇指南#

Viewer使用情境輸出依賴
ViewerGL互動開發與除錯（預設範例用）即時視窗pyglet、imgui_bundle
ViewerRTXNVIDIA GPU 上的路徑追蹤即時渲染即時視窗ovrtx、usd-core、pyglet
ViewerFile錄製供回放／分享.json／.bin 檔無
ViewerUSD整合 3D 製作管線（Omniverse／Isaac Sim）.usd 檔usd-core
ViewerRerun進階視覺化與分析（時間序列圖）網頁介面rerun-sdk
ViewerViser瀏覽器視覺化與 Jupyter notebook網頁介面、.viser 檔viser
ViewerNull無頭（headless）／自動化環境無無

共通介面（ViewerBase）#

所有 viewer 繼承 ViewerBase，共用同一套迴圈模式：

viewer = newton.viewer.ViewerGL()
viewer.set_model(model)              # 可加 max_worlds=4 只渲染前 4 個環境

while viewer.is_running():
    if viewer.should_step():         # 每 frame 呼叫一次；暫停時可用「.」單步
        simulate_substeps()          # 推進模擬
    viewer.begin_frame(sim_time)
    viewer.log_state(state)          # 更新剛體變換、粒子位置
    viewer.end_frame()

viewer.close()
  • 核心迴圈：set_model()、begin_frame()、log_state()、end_frame()、is_running()、is_paused()（ViewerGL 以空白鍵切換）、should_step()、close()。
  • 相機與排版：set_camera()（位置、pitch、yaw）；set_world_offsets() 把多個世界以網格間距排開（物理保持在原點、只有視覺偏移——與 replicate(spacing=…) 相比更穩定）。
  • 除錯疊加：log_lines()（射線、法線、力向量）、log_points()（點雲）、log_contacts()（接觸法線）、log_gizmo()（變換 gizmo）、log_scalar()／log_array()（數值序列，Rerun 可畫時間圖）、log_image()（ViewerGL 的可停靠影像視窗，適合顯示 TiledCamera 輸出）。
  • RL 訓練提示：大量平行環境時用 max_worlds 限制渲染數量以免拖慢訓練。

錄製與離線輸出#

# 錄製到 JSON（人類可讀、零依賴）
viewer = newton.viewer.ViewerFile("recording.json")
# ... 模擬迴圈 ...
viewer.close()   # 關閉時存檔

# 輸出時間取樣 USD，供 Omniverse / Isaac Sim 開啟
viewer = newton.viewer.ViewerUSD("output.usd")
# ... 模擬迴圈 ...
viewer.close()

錄製檔可重新載入回放，並能還原 model 與 state。

網頁檢視器：Rerun 與 Viser#

  • ViewerRerun：預設在本機啟動 Rerun 檢視器，也可指定遠端伺服器位址；擅長時間序列數據分析。
  • ViewerViser：啟動網頁伺服器（預設 http://localhost:8080），支援 .viser 錄製檔與 notebook 內嵌時間軸播放——Jupyter 工作流程首選（pip install "newton[notebook]"）。
